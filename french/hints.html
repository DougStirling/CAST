Navigateurs

CAST fonctionne avec les versions d'Internet Explorer, Firefox et Safari au moment de la rédaction de ce (mi-2016), mais la version basée sur le navigateur de CAST cessera de fonctionner dans tous les navigateurs à un moment donné dans l'avenir. Nous recommandons donc que CAST est téléchargé et la version stand-alone de CAST est utilisé à la place.

Une version récente de Java est nécessaire pour afficher les diagrammes interactifs CAST. Quand un CAST e-book est démarré, une vérification est faite pour savoir si Java est installé; si votre ordinateur ne parvient pas la vérification Java, les versions "complets" de pages ne seront pas affichés. (Vous pouvez cependant toujours être en mesure de parcourir les versions sommaires et vidéo des pages si l'e-book a eux.

(Malheureusement, Java ne peut pas être installé sur les ordinateurs tablettes afin que les diagrammes interactifs ne peuvent pas être utilisés sur eux.)

Les différentes versions des pages

Dans certaines des pages au sein du CAST e-books, vous verrez un petit triangle en haut à droite. Ceci est un menu dans lequel vous pouvez Dispay versions alternatives de la même page -- peut-être avec un agent biologique, d'affaires ou par exemple climatiques.

Pour conférenciers

Les acteurs principaux des e-books ont été conçus pour l'auto-étude, mais les diagrammes interactifs sont également utiles pour expliquer des concepts à des conférences. Pour rendre cela plus facile pour les enseignants, trois spéciales e-books ont été créés ne contenant que les diagrammes interactifs (applets) de l'introduction d'e-livres sans texte qui l'entoure. Les diagrammes sont plus grandes que celles de l'étudiant d'e-livres et un bouton est prévu pour donner des conseils sur l'utilisation de chaque diagramme.

Pour utiliser le sermonner les e-books, sélectionnez l'onglet e-livres puis cliquez sur Applets pour conférenciers.

Il est possible d'inclure des hyperliens dans une présentation PowerPoint pour afficher directement les pages CAST spécifiques. Une présentation PowerPoint a été préparée qui décrit comment le faire et donne des exemples.

Table des matières

Dans chacun des CAST e-livres, un bouton sur la droite de la bannière donne accès aux versions des chapitres de livres électroniques qui ont été formatés de différentes façons pour l'impression.

Vous pouvez afficher une liste des sections et des pages à l'intérieur de chaque chapitre, éventuellement avec une brève description du contenu des pages. Vous pouvez également afficher toutes les pages dans le chapitre ensemble comme une seule «live» page HTML. (Notez cependant que cette contient toutes les applets dans les chapitres et peut prendre un certain temps à charger.) Dans certains e-books, des versions plus courtes de synthèse des pages ont été préparés avec les applets remplacé par des diagrammes statiques. Si des versions sommaires des pages d'un chapitre existent, il est également possible d'afficher une version résumée de l'ensemble du chapitre.

Chacune de ces versions d'un chapitre est affichée dans une fenêtre du navigateur et peut être imprimé à partir de là en utilisant la commande d'impression du navigateur. Dans certains e-books, pré-formaté versions PDF des chapitres peut également être consulté.

Personnalisée et traduit les e-livres

Il est possible de créer CAST e-books avec une sélection différente et commande de matériel. Par exemple, l'auteur a produit des e-livres pour accompagner divers cours à sa propre université et pour les groupes extérieurs et les entreprises.

Un programme et des documents ont été écrits pour simplifier le processus de personnalisation. Un e-book personnalisé peut être créé avec seulement ce programme et un éditeur HTML.

De manière similaire, il est possible de créer CAST e-books dans d'autres langues, même si le processus est un peu plus complexe pour une traduction complète, y compris les diagrammes statiques et interactives.

Si vous êtes intéressé à créer un personnalisé ou traduit livre électronique, s'il vous plaît contactez l'auteur.
