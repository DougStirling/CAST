Advantages de téléchargement

Bien CAST peut être utilisé directement à partir de le CAST site web, nous encourageons les utilisateurs à télécharger CAST et l'exécuter à partir d'un disque dur local.

  • CAST peut être lent lors de l'exécution de plus de connexions internet pauvres.
  • Exécution de CAST partir d'une copie locale réduit la charge sur le CAST serveur web.
  • Une copie téléchargée peut être utilisé sur un ordinateur sans connexion Internet.

Plus important encore est que les diagrammes interactifs et dynamiques qui sont une partie essentielle de CAST ne sont actuellement pas pris en charge par certains navigateurs web et finira par cesser de travailler dans d'autres.

Téléchargement et installation

La collection française de CAST e-books est fourni en fichier 'zip' compressé. En cliquant sur ce lien télécharge un fichier appelé "cast_for_africa_installer.zip". Vous devez copier ce fichier à partir de votre dossier de téléchargement par défaut à l'endroit où vous voulez CAST pour être installé - votre bureau peut être l'endroit le plus pratique.

Comment extraire CAST partir de ce fichier dépend de votre système d'exploitation.

Windows
Cliquez droit sur le fichier zip puis sélectionnez la commande Extract all dans le menu contextuel qui apparaît. Vous serez demandé où vous voulez CAST installée. Cela crée un dossier appelé CAST sur votre disque dur contenant tous CAST. (Notez que simplement en double-cliquant sur le fichier zip va vous montrer les fichiers qui sont en elle, mais ne sera pas les extraire sous une forme qui permet CAST pour être utilisé — vous devez utiliser la commande xxxx Extract all.)
Macintosh
Sur un Mac, il suffit de double-cliquer sur le fichier zip; CAST sera élargi dans un dossier appelé CAST dans le même emplacement que le fichier zip.

Dans les deux cas, vous vous retrouverez avec un dossier appelé CAST sur votre disque dur. Le fichier zip se peut maintenant être supprimé.

Java

Vous aurez besoin d'installer le support de Java sur votre ordinateur, à la fois pour exécuter la version stand-alone de CAST et aussi pour obtenir l'accès aux diagrammes interactifs dans le version.You par navigateur doit aller sur le site Java Web principal d'Oracle et de faire en sorte que vous avez la version la plus récente de Java installé.

Exécution de CAST à partir d'une copie téléchargée

Au niveau supérieur de votre dossier CAST installé, vous devriez voir un fichier appelé "Start_CAST.jar". Si le support de Java a été installé, double-cliquant sur ce fichier doit démarrer la version CAST autonome.

Si vous souhaitez continuer à utiliser la version basée sur le navigateur de CAST (non recommandé), vous pouvez double-cliquer sur le fichier appelé "collection_french.html"; cela devrait montrer cette version de CAST dans votre navigateur par défaut. (Il sera identique à la fenêtre que vous consultez actuellement à moins qu'il est une version plus récente de CAST).
