Important statistical skills

Statistical methods are always used in the context of a problem in some applied discipline such as medicine, business, economics, ecology, etc. Important skills are:

  • collection of information that is relevant to the problem,
  • selection of an appropriate way to analyse the data, and
  • using the results of the analysis to answer questions in the context and terminology of the application area.

A further practical skill that is important to statisticians is gaining expertise in using a statistical program to analyse data.

Exercises involving external analysis of data

In practice, statisticians must use a statistical program to analyse data. The exercises in this e-book involve exporting data from CAST and performing some analysis in such a program. The data are exported in CSV format that can be imported into most statistical programs.

These exercises both help to reinforce statistical concepts and give practice with using a statistical program to analyse data.
