Chapter 4   Categorical Variables

4.1   Frequency tables

4.1.1   Proportions for categorical data

In this exercise, you should produce a frequency table for a data set in an external statistics program. The exercise asks you to find the proportion or percentage of values equal to, not equal to, or greater than a particular categorical value.

4.2   Bar and pie charts

4.2.1   Bar charts

Bar charts display categorical data graphically. In this exercise, you should draw bar pie charts for four data sets and match them to the data set names.

4.2.2   Pie charts

Pie charts display the same information in a different way. This exercise asks you to match pie charts for four data sets to their data set names.

4.3   Contingency tables

4.3.1   Proportions from contingency tables

In this exercise you should produce a contingency table then find a joint, marginal or conditional proportion.
