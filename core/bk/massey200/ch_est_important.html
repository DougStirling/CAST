What you need to know in this chapter

You should concentrate on the following material when studying the chapter about estimation.

4.1 Estimation

You need to know the definitions of an estimator's bias and standard error and how they are combined to give its mean squared error. All you need to know about consistency is that it arises when both the bias and standard error approach zero in large samples.

4.2 Method of moments

You should be able to apply the method of moments for estimation in any distribution with a single unknown parameter.

4.3 Maximum likelihood

This is a core section of the e-book. Many topics later in the course are based on the likelihood function and maximum likelihood estimation, so make sure that you thoroughly understand the concepts. You should be able to apply maximum likelihood to any distribution with a single unknown parameter.

4.4 Asymptotic properties of MLEs

You should remember that maximum likelihood estimators are asymptotically unbiased and normally distributed, but the asymptotic formula for the variance in page 4.4.1 does not need to be remembered. What is more important is the method of finding the MLE's standard error on page 4.4.2 from the second derivative of the log-likelihood and you should be able to apply this to find the standard error of any single unknown parameter.

4.5 Numerical methods for MLEs

You should be able to apply the Newton-Raphson algorithm to find a parameter's maximum likelihood estimator and its standard error for any distribution.
