Chapter 1   Introduction: Des Données

1.1   Contexte

1.1.1   Importance of context

L'accent de la statistique est de répondre aux questions qui sont exprimés dans la langue de certains domaines d'application. Méthodes statistiques pour l'analyse des données sont un élément central de statistiques, mais le contexte des données est plus important.

1.1.2   Statistiques

L'analyse statistique est un processus qui consiste à identifier les questions d'intérêt, la collecte des données et des analyses et produire un rapport. Dans la vie réelle des problèmes, les étapes de collecte de données et d'analyse peut être répétée plusieurs fois.

1.1.3   The statistical process

Dans de nombreuses applications, le cycle de collecte de données et d'analyse est un élément central de la quête de l'amélioration des systèmes et des processus.

1.2   Structures de données standard

1.2.1   Variables and individuals

La plupart des ensembles de données contiennent une ou plusieurs mesures de chacun d'une collection des «individus» (aussi appelé «cas» ou «unités»).

1.2.2   Types of variable

Les variables sont classées en variables numériques et catégorielles. Une classification plus fine est également esquissé.

1.2.3   Les variables catégorielles et les groupes

Une variable catégorique peut être utilisé pour diviser les «individus» en groupes. De façon équivalente, les données groupées peuvent être représentés dans une matrice de données avec une variable catégorielle.

1.2.4   Définir de nouvelles variables

Parfois, un rapport ou une différence de deux variables dans une matrice de données est plus facile à interpréter que les variables d'origine.

1.2.5   Time-ordered data

Dans certains matrices de données, les lignes sont dans un ordre chronologique.

1.2.6   Multi-level data ((avancé))

Parfois, l'information est disponible à la fois du groupe et au niveau individuel -- multi-niveau des données. Ces données sont le plus naturellement stockées dans des matrices de données à deux.

1.2.7   Structure de données et la méthode statistique

L'analyse statistique est spécifique à la structure des données (ie le type de variable dans la matrice de données). CAST commence avec des méthodes descriptives pour explorer les données, il passe ensuite à des méthodes déductives qui tiennent compte de l'aléatoire dans les données.

1.3   Variation

1.3.1   Signal et le bruit

Dans de nombreuses situations, l'information (signal) peut être masquée par des variations aléatoires (bruit).

1.3.2   La variabilité naturelle

Lorsque les données sont collectées à partir des «individus», ils varient souvent considérablement.

1.3.3   Variabilité causée par des expériences

Différences intentionnelle à des conditions expérimentales peuvent aussi causer des différences systématiques dans les variables. La variabilité naturelle rend plus difficile d'interpréter les résultats expérimentaux.

1.3.4   La variabilité des données d'enquête

La variabilité naturelle des individus rend également plus difficile à interpréter l'information à partir d'enquêtes.

1.3.5   Explained and unexplained variation

Une certaine variation dans une variable peut être expliquée en termes d'autres variables enregistrées. Autre variation est le résultat de la variabilité naturelle dans les individus.

1.3.6   Prédire les valeurs futures

Variation dans un ensemble de données peuvent nous aider à prédire les valeurs qui pourraient se produire si d'autres données du même genre sont collectées à l'avenir.
