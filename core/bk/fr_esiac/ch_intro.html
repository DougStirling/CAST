Dans un large éventail de professions, les décisions sont tenus et les conclusions doivent être prises à propos d'autres plans d'action. Les données doivent être collectées pour jeter la lumière sur ces problèmes. Statistiques est un ensemble de techniques qui fournissent des informations utiles aux décideurs de ces données.

  • Les écologistes utilisent des données sur l'abondance des espèces pour évaluer les changements dans les habitats.
     
  • L'Organisation des Nations Unies Objectifs du Millénaire pour le développement fournir des cibles pour améliorer la santé, l'éducation et l'environnement dans les pays en développement. Les données sont nécessaires pour surveiller la réalisation de ces objectifs.
     
  • Les données climatiques sont analysés pour aider à recommander des pratiques agricoles.
     
  • Les décisions d'affaires sont de plus en plus fondées sur des données plutôt que «intuition».
     
  • Les gouvernements et autres décideurs sont tenus de fonder leurs actions sur des preuves à partir des données.
     
  • Les traitements médicaux sont basées sur les résultats des essais de médicaments et évaluation objective des thérapies alternatives.
     
  • ...

Bien qu'il y ait beaucoup à dire pour les statistiques de l'enseignement dans le contexte d'un domaine d'application qui est pertinente pour la carrière de la personne qui apprend le sujet, analyse statistique des données est basée sur la même approche et «boîte à outils» de base des techniques dans tous les domaines.

En enseignant les statistiques comme une discipline à part entière, nous soulignons cette approche commune à l'utilisation de données pour obtenir des informations sur les problèmes de la vie réelle.
