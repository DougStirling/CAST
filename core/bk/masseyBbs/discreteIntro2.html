Bar charts for discrete distributions

The main way to graphically display categorical data sets is with a bar chart in which the bar heights give the proportion of times the different categories arise in the data set.

A similar bar chart can be used to describe random categorical measurements with the bar height equal to the probabilities of the categories.

Similar bar charts can be used to describe discrete numerical variables, with the bars rising from a standard numerical axis.

Empirical probabilities

In most practical situations, we cannot obtain probabilities for the different possible values of a discrete measurement by reasoning alone. We must use empirical probabilities instead — the proportion of times the different values arise in a random sample.

If these empirical probabilities are based on a large enough sample, they will be close to the true underlying 'infinite sample' probabilities, as described earlier.
