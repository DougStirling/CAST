In data sets with only a single variable — a single measurement from each individual (or unit) — the best prediction of another individual's value would be the sample mean.

If we have more information about each individual — a data matrix with more than one variable — then we might be able to use the values of these extra variables to get a more accurate prediction. For example, a person's age, gender and years of education, might be used to predict that person's income.

This chapter presents methods for predicting the value of one variable from known values of the others.
