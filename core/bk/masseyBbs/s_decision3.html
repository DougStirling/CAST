Significance level

There is always a trade-off between the probabilities of Type I and Type II errors and selecting a critical value to reduce one will increase the other. The probability of a Type I error is often fixed at 5% or 1% — the significance level of the test.

If the significance level of the test is set to 5% and we decide to reject H0 then we say that H0 is rejected at the 5% significance level.

P-values and decisions

There is a simple way to reach a decision about rejecting H0 based on the test's p-value. For a test with significance level 5%:

Decision
p-value > 0.05     accept H0
p-value < 0.05     reject H0

For a test with significance level 1%, H0 is rejected if the p-value is less than 0.01.

If computer software provides the p-value for a test, it is easy to translate it into a decision to accept (or reject) the null hypothesis at the 5% or 1% significance level.
