Response distribution at each X

A regression model for the response in a bivariate data set describes how the response distribution depends on X.

The changes in the distribution of Y for different values of X are most easily visualized when we have several observations of Y for each value of X. This frequently happens with experimental data, where we control the value of X, or if X takes on discrete values, as in the following example.

House prices and bathrooms

The sale prices of all houses sold in an area were collected. How does the sale price relate to the number of bathrooms in the houses?

Normal linear model for the response

The most commonly used regression model is a normal linear model. This model involves:

Normality
At each value of X, Y has a normal distribution.
Constant variance
The standard deviation of Y is the same for all values of X.
Linearity
The mean of Y is linearly related to X.

The last two properties of the normal linear model can be expressed as

σy  =  σ

μy  =  β0  +  β1x
