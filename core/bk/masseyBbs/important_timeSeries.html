What you need to know

You will not be examined about everything in this chapter. Some of the material has been included to explain why the chapter's methods are used, in the hope that it will help you to understand these methods better. What you need to learn for the exam is more limited.

We now describe what we expect you to be able to do in the exam after studying the time series chapter.

A.   Time series

A time series describes how some quantity varies over time.

1.   Name the four components of a time series
Trend, seasonal, cyclical, random.
2.   Describe the pattern in a time series plot
Identify which of the four components are present and describe them.
  • If there is trend, is it positive or negative? Linear or nonlinear?
  • If there is a seasonal pattern, which 'seasons' are high and low?
3.   Smoothing
(a)   Moving averages
  • Evaluate a moving average with odd run length.
  • Evaluate a moving average with even run length.
(b)   Exponential smoothing
  • In a table containing the original time series values and some exponentially smoothed values, evaluate a missing exponentially smoothed value.
  • Explain how the smoothing constant affects the smoothed values.
  • Use the last exponentially smoothed value to forecast one time period into the future.
4.   Linear trend
(a)   Explain why "time" is often recoded with one time as X=0 before fitting a linear trend:
  • for seasonal data,
  • for interpretation of the intercept, and
  • to avoid numerical problems
(c)   Given Excel output from a linear regression model
  • Write down the model.
  • Assess whether there is trend (from the p-value associated with the slope).
  • Use the model to forecast a future value.
  • Discuss whether the linear forecast is better than one from exponential smoothing for a specific data set.
 

B.   Index numbers

Index numbers are a special kind of time series that summarise how prices (and sometimes other quantities) vary in relation to a base time.

1.   Simple price index
From a table of prices,
  • Find the index number of any year, using a specified base time.
  • If the table includes index numbers, identify which time is the base time.
2.   Aggregate price index
  • Explain how an aggregate price index summarises changes in a 'basket' of items.
