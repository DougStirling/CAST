Patterns in the structure of data

The statistical methods that can be used to analyse a data set depend on its internal structure. The same data structure can arise in many different contexts.

Variables and 'individuals'

Data sets can have a complex structure, but most consist of measurements called variables made from each of a collection of individuals.

Univariate, bivariate and multivariate data

Data sets with only one variable (measurement) are called univariate. When there are two or more variables from each individual, the data are multivariate. Bivariate data is a special case where there are only 2 variables.
