Probability of getting one of several values

When one value is sampled from a population, the probability that it is x is the proportion of population values that equal x. In a similar way,

The probability that a single sampled value is either x, y, ... is the proportion of population values that are either x, y, ... .

If 12 values in a population of 100 values are under 3.5, the probability that a single sampled value will be under 3.5 is

Prob( X < 3.5 )   =  12/100 = 0.12

More generally,

Prob( a < X < b )   =   propn of values between a and b.
