Dans les chapitres précédents, nous avons décrit les modèles des ensembles de données univariées et développé intervalles de confiance et tests d'hypothèses pour les paramètres de ces modèles.

Dans ce chapitre, nous décrivons les modèles pour les ensembles de données bivariées dans lequel une réponse dépend d'une variable explicative numérique - le type de données que nous avons résumé plus tôt avec une ligne moins carrée. Modèles de régression linéaire sont étroitement liées aux lignes carrés moins.

Les paramètres de ces modèles de régression sont souvent d'une importance pratique, et l'inférence sur leurs valeurs nous aident à interpréter la relation entre les deux variables.
