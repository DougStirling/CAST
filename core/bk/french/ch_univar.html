Les données peuvent avoir une structure riche, mais nous concentrer d'abord sur la structure simple de données, une mesure numérique unique à partir de chaque «individu» dans un groupe. Nous appelons les données de cette forme d'un lot de numéros, ou d'un ensemble de données numériques univariées.

Des exemples des types de «individus» à partir de laquelle nous avons souvent des données enregistrées se trouvent ...

  • Personnes
  • Les plantes
  • Les échantillons de sortie à partir d'un processus de fabrication
  • Admissions de patients à l'hôpital

Dans ce chapitre, nous décrivons plusieurs moyens communs pour afficher la variabilité dans un lot de données numériques. Nous montrons comment ces données peuvent fournir des informations utiles.
