Chapter 6   Multivariée des données

6.1   Affichage des données multivariées

6.1.1   Representing a third variable

Une troisième variable numérique peut être représentée dans un diagramme de dispersion en utilisant des symboles ou des couleurs différentes.

6.1.2   Rotating 3D scatterplots

Trois variables numériques peuvent être affichées dans un nuage de points en 3 dimensions; cela peut être tourné pour aider à comprendre les relations entre les données.

6.1.3   Scatterplot matrix and brushing

Un tableau de nuages de points de toutes les paires de variables est souvent instructif, surtout si les nuages de points sont liés dynamiquement.

6.1.4   Brushing example

«Brossage» se réfère à la mise en évidence dynamique des mêmes personnes à plusieurs écrans liés.

6.1.5   Slicing

Tranchage est une autre technique dynamique. Seules les observations dans une plage de valeurs d'une variable (une tranche) sont affichés dans les affichages liés.

6.2   Groupes et régression

6.2.1   Additional variables in regression

Corrélation des moindres carrés et sont utilisés pour décrire la relation entre les deux variables numériques. Des mesures supplémentaires de chaque individu peuvent potentiellement aider à affiner notre compréhension de la relation.

6.2.2   Displaying groups

Différents symboles ou des couleurs peuvent être utilisés pour représenter une troisième variable catégorique dans un nuage de points.

6.2.3   Regression with grouped data

La relation entre Y et X peut être décrit séparément par une ligne des moindres carrés à l'intérieur de chaque groupe. Cela devrait conduire à une meilleure prédiction de la réponse si la relation est différente dans les différents groupes.

6.2.4   Parallel regression lines

Si les lignes de régression pour les différents groupes sont parallèles, il est facile de résumer les différences entre les groupes numériquement et interpréter ces différences.

6.2.5   Transformed variables and groups ((avancé))

Transformations peuvent linéariser la relation entre les variables de réponse et explicatives dans chaque groupe et de donner également des lignes de régression en parallèle.

6.2.6   Grouping with a numerical variable ((facultatif))

Une variable numérique peut être utilisé pour séparer les individus en groupes.

6.2.7   Scatterplot matrix with groups

Les groupes peuvent aussi être représentés par différents symboles ou couleurs sur une matrice de diagramme de dispersion qui décrit les relations entre 3 ou plus d'autres variables.

6.3   Régression multiple

6.3.1   More than one explanatory variable

Dans de nombreux ensembles de données, deux ou plusieurs variables explicatives pourraient potentiellement affecter la réponse. L'utilisation de deux ou plusieurs variables explicatives peuvent donner des prévisions plus précises.

6.3.2   Multiple regression equation

Un modèle linéaire simple avec une seule variable explicative peut être étendue avec des termes supplémentaires pour expliquer l'effet supplémentaire d'autres variables explicatives.

6.3.3   Interpreting coefficients

Le coefficient de pente associée à une variable explicative décrit son effet, si toutes les autres variables sont maintenues constantes. Elle peut avoir un signe différent du coefficient de corrélation entre la variable et la réponse.

6.3.4   Scatterplot of three variables

La relation entre une variable de réponse et deux variables explicatives peut être effectivement affiché dans un diagramme de dispersion rotatif en 3 dimensions.

6.3.5   Displayng the linear equation for Y vs X and Z

L'équation d'un modèle linéaire Y en fonction de X et Z peut être affichée comme un plan en 3 dimensions.

6.3.6   Fitted values and residuals

Les résidus sont des distances verticales des croix sur un nuage de points en 3 dimensions au plan représentant le modèle.

6.3.7   Least squares estimation

Une méthode d'estimation objectif est de minimiser la somme des carrés des résidus - le principe des moindres carrés.

6.4   Relations marginaux

6.4.1   Misleading marginal correlation

Variables Y et X peuvent être corrélés positivement dans l'ensemble, mais ont zéro ou même corrélation négative à chaque valeur d'une variable catégorique, Z. La variable Z est appelé (ou caché) variable de cache.

6.4.2   Misleading marginal difference in means

Une variable qui se cache peut aussi fausser la différence entre les moyens de Y en deux groupes (ie pour deux valeurs d'une variable catégorique, X).

6.4.3   Simpsons paradox

Si X, Y et Z sont tous catégorique, un renversement de la relation marginale entre X et Y et de leurs relations avec sursis pour les différentes valeurs de Z est appelé le paradoxe de Simpson.

6.4.4   Other examples with lurking variables

Quelques exemples supplémentaires sont présentés où une variable cachée, Z, peut conduire à une conclusion trompeuse de la relation marginal. Une analyse complète en utilisant Z est toujours plus complexe mais il est essentiel de comprendre la relation.
