Chapter 10   Tester des hypothèses

10.1   Introduction aux tests d'hypothèses

10.1.1   Inference

Estimation demande qui valorise pour un paramètre inconnu sont compatibles avec les données qui ont été recueillies. Test d'hypothèse demande généralement si les données sont compatibles avec certaines valeurs de paramètres.

10.1.2   Soccer league simulation

Certaines équipes dans une ligue de soccer sont mieux que d'autres, ou est le tableau de la ligue de fin de saison en accord avec des résultats aléatoires des équipes aussi assortis? Une simulation peut aider à répondre à cette question.

10.1.3   Simulation to test a proportion

Cette page présente une simulation pour tester si le nombre d'observations observées dans une catégorie est compatible avec une probabilité donnée.

10.1.4   Test for a mean

Cette page utilise une simulation pour tester si un échantillon est compatible avec une moyenne de population particulier.

10.1.5   Randomisation tests ((facultatif))

Un procédé est présenté pour déterminer si la population des moyens dans deux groupes sont identiques.

10.1.6   Soccer league randomisation ((avancé))

La corrélation entre les points finaux des équipes en deux saisons successives sera relativement élevé si certaines équipes sont mieux que d'autres. Les points de randomisation dans la deuxième saison donne une indication de savoir si la corrélation réelle est anormalement élevée.

10.1.7   Common patterns in hypothesis testing

Tous les tests d'hypothèses impliquent une hypothèse nulle, une statistique sommaire pour les essais, une hypothèse alternative et une valeur p.

10.2   Test d'hypothèse pour les proportions

10.2.1   Inference about parameters

Lorsque les données sont un échantillon aléatoire, les hypothèses nulle et alternative sont généralement exprimés en termes de paramètres de la population. Pour les échantillons catégoriques, les hypothèses se réfèrent aux probabilités des catégories.

10.2.2   P-value for testing proportion

La valeur p pour tester une proportion est la probabilité d'obtenir un tel nombre «extrême» de succès lorsque l'hypothèse nulle est titulaire. Il peut être trouvé à partir de la distribution binomiale.

10.2.3   Another example of testing a proportion

Un exemple montre comment utiliser une distribution binomiale pour trouver la valeur p pour un test.

10.2.4   One- and two-tailed tests

Si les valeurs dans les deux extrémités de la distribution binomiale soutiennent l'hypothèse alternative, la probabilité de queue doit être doublé pour donner la p-valeur pour le test.

10.2.5   Tests based on normal approximation

Lorsque la taille de l'échantillon est grande, une approximation normale de la distribution binomiale peut être utilisée pour trouver le p-valeur pour un test d'hypothèse.

10.2.6   Statistical distance

La distance statistique entre une estimation et la valeur de paramètre hypothétique est la différence divisée par l'erreur-type de l'estimation. Il doit avoir environ une distribution normale standard si l'hypothèse nulle est vraie.

10.2.7   Tests based on statistical distance

La valeur p pour tester une proportion peut être évaluée comme une zone de queue d'une distribution normale standard correspondant à des valeurs plus «extrêmes» que la distance statistique entre p et π.

10.3   Tests d'hypothèses sur les moyens

10.3.1   Introduction

Pour les populations numériques, les hypothèses nulle et alternative précisent généralement les valeurs de la moyenne de la population. Les essais sont basés sur l'échantillon moyen.

10.3.2   Test for mean (known σ)

Si l'écart type de population est connu, la répartition de la moyenne de l'échantillon peut être trouvée lorsque l'hypothèse nulle est vraie. La p-valeur pour le test est une zone de queue de cette distribution.

10.3.3   P-value from statistical distance

La p-valeur est plus facile à trouver à partir de la «distance statistique» entre l'échantillon et des moyens hypothétiques. La p-valeur peut être trouvée en tant que zone de queue de la distribution normale standard et est à peu près correcte, même si la répartition de la population est pas normal.

10.3.4   The t distribution

Lorsque l'écart type de population est inconnu, une statistique de test différent doit être utilisé. Il a une distribution standard connu sous le nom à la distribution.

10.3.5   The t test for a mean

La distribution t est utilisé pour obtenir une p-valeur pour les essais environ une moyenne de population lorsque l'écart type est inconnu.

10.4   Décisions et le niveau de signification

10.4.1   Hypothesis tests and decisions

Dans de nombreuses applications, une autre action est prise si l'hypothèse nulle est «acceptée» ou «rejetée». Deux types d'erreurs sont possibles d'une telle décision - accepte l'hypothèse nulle alors qu'elle est fausse, ou rejette quand il est vrai.

10.4.2   Decision rules

La règle de décision est basée sur un échantillon statistique. Pour les tests sur certains paramètres, les probabilités d'erreurs Type I et Type II peuvent être calculés.

10.4.3   Significance level and p-values

De nombreux tests sont effectués avec une probabilité pré-spécifiée d'une erreur de type I - le niveau de signification. L'hypothèse nulle est rejetée si le p-valeur pour le test est inférieur au seuil de signification.

10.4.4   Sample size and power

La puissance d'un test d'hypothèse est un moins la probabilité d'une erreur de type II. A un niveau de signification fixe, ce qui augmente la taille de l'échantillon permet d'améliorer le pouvoir de l'essai.

10.5   Propriétés des valeurs p

10.5.1   Null and alternative hypotheses

Tests d'hypothèses demandent si les exemples de données sont conformes à une déclaration sur les paramètres appelés l'hypothèse nulle.

10.5.2   P-value and consistency with null hypothesis

Une valeur de p est une mesure numérique de savoir si les données de l'échantillon sont compatibles avec l'hypothèse nulle.

10.5.3   Distribution of p-values

Lorsque l'hypothèse nulle est vraie, tout p-valeur entre 0 et 1 est également probable. Lorsque l'hypothèse alternative est vraie, les valeurs p sont proches de 0 plus probable. Un p-valeur proche de 0 donne donc la preuve que l'hypothèse alternative est titulaire.

10.5.4   Interpretation of a p-value

Une valeur de p est la probabilité d'obtenir de telles données «extrême» lorsque l'hypothèse nulle est titulaire.

10.5.5   P-values for other tests

Les valeurs de p ont les mêmes propriétés et interprétation pour tous les tests. Un test pour savoir si une population a une distribution normale est utilisée comme exemple.
