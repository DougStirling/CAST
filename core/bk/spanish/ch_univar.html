Los datos pueden tener una estructura rica, pero primero concentramos en la estructura de los datos más simples, una sola medida numérica de cada uno “individuo” en un grupo. Nosotros datos de llamada de esta forma una hornada de números, o un conjunto de datos numérico univariante.

Los ejemplos de los tipos de “individuos” de cuáles a menudo registramos datos son…

  • Gente
  • Plantas
  • Muestras de salida de un proceso de fabricación
  • Admisiones de pacientes a un hospital

En este capítulo, describimos varias maneras comunes de exhibir la variabilidad en una hornada de datos numéricos. Demostramos cómo tales datos pueden proporcionar la información útil.
