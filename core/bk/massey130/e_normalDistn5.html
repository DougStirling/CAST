Inverse problem

The easiest questions about normal distributions give you a range of x-values and ask for the probability of getting a value in this range. The inverse problem gives you a normal probability and asks for the corresponding values of X.

This type of question can be answered using normal tables. Firstly find the range of z-scores that gives the required probability, then transform this interval into the actual measurements with:

x  =  μ  +  z σ

 

In the following exercise, a standard normal table should be used to find areas under the normal curve.

Since the table only gives the area to the left of any z-score, you may need to subtract such areas to answer the question. A template is provided for this purpose.

Repeat until you have mastered the method.
