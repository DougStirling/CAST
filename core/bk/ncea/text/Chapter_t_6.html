Chapter 6   Chance and Data (1.12)

6.1   Probability

6.1.1   Finite populations

When randomly selecting one value from a population of N different values, the probability of getting any individual value is 1/N. More generally, the probability of sampling a value in any range is the proportion of population values in the range.

6.1.2   Probabilities with infinite populations

The probability of any type of value is again the proportion of such values in the population. It can also be interpreted as the limiting proportion from a sample of values if the sample size is increased indefinitely.

6.1.3   Bar charts of discrete probabilities

Infinite categorical or discrete populations can be described by bar charts of the probabilities.

6.1.4   Relative frequency and area

In a histogram, the proportion of the total area that is above any class equals the relative frequency of the class.

6.1.5   Exercise: Probabilities and finite populations

A finite categorical population is shown as a bar chart and the probability of a randomly sampled value being in a range of categories must be found.

6.1.6   Exercise: Probabilities and numerical population

This exercise is simlar to the one on the previous page but relates to a numerical population that is summarised by a histogram. The probability must be estimated roughly 'by eye' from the area under the histogram.

6.2   Contingency tables

6.2.1   Contingency tables

Categorical data collected from different groups can be shown in a contingency table. It contains a simple frequency table for each group.

6.2.2   Contingency table examples

Contingency tables sometimes arise from experiments. Data collected from surveys are often presented in contingency tables.

6.2.3   Proportions within groups

It is easier to compare groups if bar charts are drawn from the proportions within each group rather than the frequencies.

6.2.4   Categorical variables and groups

A categorical variable can be used to split the 'individuals' into groups. Equivalently, grouped data can be represented in a data matrix with a categorical variable.

6.2.5   Exercise: Create a contingency table

This exercise presents a list of values of two categorical variables; you must generate a contingency table summarising the data.

6.3   Time series plots

6.3.1   Time series plots

A time series plot is similar to a scatterplot with 'time' as the variable on the horizontal axis. Successive values are usually joined by lines to emphasise systematic movements in the series.

6.3.2   Patterns in time series

A few examples show time series with trend, seasonal pattern and other oscillations.
