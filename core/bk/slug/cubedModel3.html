Hypothesis Tests for parameters

The Minitab output also provides a column of p-values associated with the model parameters. These p-values describe whether the data are consistent with the corresponding model parameter being zero.

Is the intercept zero?
The p-value for the constant term is 0.117, meaning that there would have been probability 0.117 of getting an intercept as far from zero as 0.1652 even if the intercept for slugs in general was 0.0. Since this is fairly high, we conclude that the data are consistent with an intercept of zero.
This is reassuring since a model with zero intercept would predict a slug with length zero to also have weight zero!
Is the slope zero?
Minitab reports the p-value for (length) to be '0.000' which really means 'under 0.0005', meaning that there would have been a probability below 0.0005 of getting a slope as far from zero as our least squares estimate of 0.0124 if the slope for slugs in general was 0.0. We therefore conclude that slug weight almost certainly is related to (length).

We could continue with this analysis by fitting a model without an intercept term, an option when specifying a regression model in Minitab. However there are other serious problems that mean a different approach is required.
