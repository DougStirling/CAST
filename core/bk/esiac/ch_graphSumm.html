The variation of values for a numerical variable holds 'useful' information that we can interpret, but there are detailed aspects of the variation that do not help us to understand the context of the data — noise.

Graphical displays that separately represent the individual values in a data set often hold a distracting amount of noise, especially in large data sets. Summarising the data in ways that do not show the individual values usually highlights the useful information — signal — in the data.

Summarising the data removes some noise and highlights the signal.
