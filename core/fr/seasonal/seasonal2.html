Les moyennes mobiles

Nous avons montré dans la première section de ce chapitre que les moyennes mobiles sont efficaces pour réduire les fluctuations aléatoires dans une série chronologique.

Les moyennes mobiles peuvent également être utilisés pour éliminer un effet saisonnier, mais la longueur d'exécution est cruciale.

Pour les données saisonnières avec la longueur du cycle n, soit une moyenne de n-point mobile doit être utilisé.

Par exemple, les données quotidiennes ont une durée de cycle de 7 jours. Chaque 7-point courant moyennes signifie exactement une valeur de chaque jour de la semaine (une valeur de chaque couleur dans le diagramme ci-dessous).

Si, par exemple, les valeurs sont régulièrement samedi bas que les autres jours, les valeurs lissées pour chaque jour de la semaine comportera un samedi et seront également affectés. Les valeurs lissées pour les samedis ne seront donc pas tendance à être plus faible que les autres jours. Cela supprime le schéma saisonnier.

Même longueurs de cycle

Malheureusement, il est une complication lorsque les données saisonnières ont une longueur de cycle, même (comme les données mensuelles avec une longueur de cycle de 12) car nous n'avons pas défini ce qu'on entend par une moyenne de n-point mobile, où n est pair.
