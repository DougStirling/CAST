Une analyse plus poussée des données désaisonnalisée

La première étape de l'analyse des données est généralement saisonnière pour enlever l'effet saisonnier.

Données   =   Effet saisonnier   +   Désaisonnalisée

Cependant, nous devons maintenant examiner les données désaisonnalisée pour les caractéristiques de tendance et cycliques. En utilisant les méthodes dans les deux premières sections de ce chapitre, nous pouvons obtenir les composantes tendance et cycle. Ce qui reste après avoir enlevé ce sont les résidus.

Désaisonnalisée  =   Tendance   +   Cyclique   +   Résiduelle

Mettre ces ensemble, nous pouvons identifier quatre éléments séparés à la variation de notre série chronologique original.

Données  =   Effet saisonnier  +   Tendance   +   Cyclique   +   Résiduelle

Il est important que vous compreniez la distinction entre ces quatre composantes d'une série temporelle saisonnière:

Effet saisonnier
Pour données mensuelles ou trimestrielles, cette composante se répète exactement de la même manière chaque année.
Tendance
Cette composante des changements en douceur sur toute la période des données. Il est souvent décrit par un modèle linéaire ou quadratique.
Cyclique
Ce volet décrit pics et les creux qui durent des valeurs des données de plusieurs. La composante comprend deux cycles à long terme et à court terme d'autocorrélation de quelques valeurs adjacentes.
Résiduelle
Cette composante est ce qui reste. Il reflète «inhabituelles» valeurs de données individuelles qui ne peuvent pas être expliquées par les effets d'autres.

Les arrivées de touristes dans les îles Fidji

Vous allez utiliser le diagramme ci-dessous pour aider à comprendre les différentes composantes de la série chronologique des arrivées de touristes dans les îles Fidji.

Initialement, la ligne bleue dans la moitié supérieure du diagramme relie les points de données brutes. Cliquez sur la case Saisonnière pour supprimer l'effet saisonnier de ce schéma et l'afficher dans la moitié inférieure du diagramme. (La moyenne a été ajoutée aux effets saisonniers pour leur permettre d'être tracée sur l'axe même que les données.) Le haut diagramme montre donc les données désaisonnalisée et le fond montre les effets saisonniers.

Maintenant également cliquer sur le Tendance case. Cela supprime une tendance quadratique à partir du haut et l'ajoute à la base.

Enfin, cliquez Cycliques pour supprimer la composante cyclique du diagramme en haut et l'ajouter à la base. Ce qui reste dans le top c'est les résidus.

Sélection des composants d'une moins-un-temps leur permet d'être affichées séparément dans la moitié inférieure du diagramme.
