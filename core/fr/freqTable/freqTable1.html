Les données numériques et catégorielles

Dans un ensemble de données, une variable numérique contient un certain nombre de chaque individu. Une variable catégorique de classer chaque individu dans un des nombreux groupes. Par exemple, une enquête sur les religions avec lesquelles un groupe de 100 individus pourraient résulter d'identifier dans les 100 valeurs,

catholique, anglicane, athée, anglicane, musulmane, ...

Dans de nombreux ensembles de données, les valeurs ne sont pas ordonnées de façon significative. Par exemple, les 100 personnes ci-dessus n'ont pas été interrogés dans un ordre particulier. (Si les données ont été recueillies dans l'ordre, des méthodes de séries chronologiques devraient être utilisées pour les analyser.) Nous ne considérons non ordonnée données catégoriques dans ce chapitre.

Les tables de fréquences

Un ensemble de données numériques non ordonnées détient beaucoup d'informations détaillées sur la distribution des valeurs. (Un nuage de points montre une information complète sur la distribution, si nous pouvons choisir de résumer avec un histogramme ou statistiques récapitulatives.)

En revanche, une non-ordonnée catégoriques ensemble de données contient beaucoup moins d'informations. Les fréquences pour les catégories distinctes sont au nombre de fois chaque catégorie se produit dans l'ensemble de données.

Les fréquences saisir pleinement toutes les informations sur la distribution des valeurs.

Ces fréquences sont généralement présentées comme un tableau de fréquences.

Enquête sur le riz

Dans le cadre d'une enquête auprès des producteurs de riz au Sri Lanka, 36 agriculteurs ont été sélectionnés au hasard dans 4 villages. Chaque agriculteur échantillonnés a été interrogé sur la variété de riz qu'il a utilisé et les variétés ont été classées en «Vieux», «Traditionnel» ou «Nouveau». Les 36 valeurs résultantes sont montrés catégoriques sur la gauche du schéma ci-dessous.

Pour calculer les fréquences de chacun des trois types de riz à la main, vous pourriez travailler à travers la table de valeurs, de tracer une ligne contre le nom de la catégorie appropriée pour chaque valeur (un pointage). Ces décompte serait finalement compté pour donner les fréquences.

Cliquez sur chacune des valeurs catégoriques à son tour pour illustrer comment les pointages et les fréquences sont obtenues.

La table finale de fréquences sur la droite résume l'utilisation des trois types de riz. Le tableau des fréquences contient toutes les informations sur la distribution des types de riz.

Examiner une variable à partir de nombreux

Dans les enquêtes comme l'enquête de riz ci-dessus, plusieurs mesures sont souvent enregistrées par chaque participant. Bien que l'analyse en profondeur des données serait enquêter sur les relations entre les variables, il est souvent utile d'examiner les distributions des variables une à la fois.

Enquête sur le riz

Dans l'enquête de riz qui a été décrit ci-dessus, cinq variables ont été mesurées à partir de chaque agriculteur.

  • Le nom du village (Sabey, Kesen, Niko ou Nanda)
  • La taille des exploitations (hectares)
  • La quantité d'engrais utilisé (tonnes/hectare)
  • Le type de riz (Vieux, Traditionnels ou Nouveaux)
  • Le rendement du riz (tonnes/hectare)
