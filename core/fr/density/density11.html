Les tables de fréquences

Un ordinateur est normalement utilisé pour dessiner des histogrammes, mais il est instructif d'examiner comment on pourrait être dessiné à la main. Les données sont d'abord résumées dans un tableau de fréquences.

Les classes sont définies dans l'histogramme des deux premières colonnes. Le premier de ces décrit la gamme des valeurs des données qui sont incluses dans chaque classe. La deuxième colonne étend ces gammes de donner gammes de toucher des valeurs — les classes qui seront utilisés pour attirer l'histogramme. La dernière colonne indique les fréquences pour les classes — le nombre de valeurs dans chaque classe.

Pourvu toutes les classes ont la même largeur, la hauteur des rectangles sont donnés par l'histogramme des fréquences.

Des précautions doivent être prises lors de la traduction de la colonne de valeurs données dans les classes. Pour admissions à l'hôpital les données, les valeurs sont les âges, de sorte correspond '40 'à n'importe quel âge entre 40 et 41.

Dans la plupart des autres types d'ensembles de données, les valeurs enregistrées sont arrondis, plutôt que tronquée, et les frontières de classe devraient refléter cette réalité. Valeurs arrondies ne doivent jamais coïncider avec les frontières de classe.

Par exemple, le Steel Works de données scories contiennent des valeurs qui sont arrondis à une décimale (6.1, 1.4, ...) et donc la valeur 6.1 pourrait être n'importe où dans la plage de 6.05 à 6.15. Un histogramme peut être tirée de la table de fréquence ci-dessous.
