Dans les ensembles de données volumineux, jittering les croix permet de montrer la densité

Une intrigue simple point est souvent suffisante pour petits ensembles de données. Cependant, dans les ensembles de données plus importants, les croix se chevauchent souvent. En effet, si plusieurs croisements coïncident, ils deviennent indiscernables d'un seul croisement, donc à haute densité peuvent être obscurcies.

Une solution consiste à déplacer le hasard traverse perpendiculairement à l'axe afin de les séparer quelque peu. C'est ce qu'on appelle les points de sautillement.

(Vous devez compter sur un ordinateur pour faire le sautillement pour vous, mais cela pourrait être fait à la main en faisant rouler un 6 ou 10 faces meurent pour chaque croix pour déterminer son sautillement en millimètres.)
