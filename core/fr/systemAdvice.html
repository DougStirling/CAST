Java

Les diagrammes interactifs dans CAST sont des programmes Java. Depuis Java ne peut pas être utilisé sur Android ou Apple comprimés, les versions interactives de pages CAST ne sont donc pas disponibles pour vous.

Bien que vous serez en mesure de voir les pages dans cet e-book, leurs diagrammes interactifs sera vide. Depuis CAST a été structuré pour expliquer des concepts statistiques en utilisant ses des diagrammes interactifs, vous ne trouverez pas, il est facile à lire.

Depuis cet e-book a des versions vidéo ou un résumé de ses pages, vous serez en mesure de les lire, mais les versions interactives des pages ne seront pas disponibles.

Depuis les versions interactives des pages CAST sont telles une partie importante de l'e-book, nous recommandons qu'il est étudié sur un ordinateur complet (portable ou de bureau) plutôt que d'une tablette ou un smartphone.

Le matériau interactif dans CAST repose sur Java, il est donc important d'installer et l'activer.

Réglages de Safari

Lors de l'exécution CAST partir d'une copie installé localement, quelques modifications doivent être apportées aux réglages de Safari avant il affiche des diagrammes interactifs de CAST (applets Java).

  • Sélectionnez Preferences à partir de Safari menu Safari.
  • Dans les paramètres Avancé, sélectionnez Afficher Développer menu dans la barre de menu. Safari va maintenant ajouter un menu supplémentaire appelé Développer à la barre de menu.
  • Dans le menu Développement, l'option Désactiver les restrictions de fichiers locaux doit être réglé pour les diagrammes interactifs à comparaître.

Par défaut, Safari affiche un avertissement pop-up quand vous regardez une page CAST contenant un schéma interactif, vous demandant si vous souhaitez Autoriser ou Bloquer il.

  • Cliquez sur Autoriser.
  • Sélectionnez Preferences à partir de Safari menu Safari.
  • Dans lee paramètres Sécurité, assurez-vous que Permettre Plug-ins est cochée, puis cliquez sur Gérer les paramètres du site.
  • Dans la boîte de dialogue qui apparaît, il ya une entrée vide qui représente fichiers installés localement. De son menu, sélectionnez Exécuter Toujours, puis modifier ses paramètres de Exécuter en mode non sécurisé.
  • Dans la boîte de dialogue qui apparaît, il ya une entrée correspondant au site Web de CAST. Changer ses paramètres pour Toujours autoriser.

Démarrage du matériel interactif

Vous devriez prochain démarrage d'un programme Java qui fait partie de l'installation de Java téléchargé; il peut être trouvé dans le dossier "CAST/core/java" et est appelé "coreCAST.jar". Un double-clic doit démarrer ce programme. Il peut être utilisé pour afficher du matériel interactif de CAST et les instructions d'utilisation associés.

Usage

Tout d'abord sélectionner cet e-book dans le menu pop-up qui apparaît. Après un court délai, une fenêtre devrait apparaître avec des menus pop-up pour vous permettre de sélectionner un chapitre, section et page dans le e-book. Les pages du navigateur dans le CAST principal e-book vous dirigeront vers les schémas appropriés à ce programme d'études.

Malheureusement, le programme Java qui est nécessaire pour faire fonctionner les diagrammes interactifs CAST vont actuellement pas fonctionner lorsque CAST est utilisé à partir d'un site Web distant. Vous devez télécharger et installer CAST pour une utilisation sur un disque dur ou serveur local.
