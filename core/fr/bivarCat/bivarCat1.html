Groupes et variables explicatives

Il a été expliqué précédemment que les données de différents groupes peuvent être combinés dans une seule matrice de données avec une variable catégorique qui donne appartenance au groupe. De façon similaire, une variable catégorique peut être utilisé pour diviser un ensemble de données en groupes.

Dans certains ensembles de données, une variable catégorielle peut être considéré comme une réponse dont les valeurs sont pensé à dépendre d'un deuxième variable catégorielle - une variable explicative. Nous pouvons alors penser que la variable explicative définissant les différents groupes et se demander comment la distribution de réponse diffère entre les groupes.

Ne pas utiliser la variable de réponse pour définir les groupes.

Si une variable catégorielle est une réponse et l'autre est une variable explicative, les méthodes dans la section précédente peut être utilisée pour voir comment la variable explicative affecte la réponse.

Des données bidimensionnelles, sans une variable explicative

Pas tous les ensembles de données ont des variables qui peuvent être classés comme une réponse et une variable explicative. Parfois, la relation entre les variables est plus symétrique, mais nous voulons toujours de découvrir si des valeurs particulières d'une variable sont associées à des valeurs de l'autre.

Pour les variables numériques, nous devrions utiliser un coefficient de corrélation pour décrire la force de la relation (par opposition aux moindres carrés pour des variables qui peuvent être classées comme une réponse et variables explicatives). Lorsque les deux variables sont catégoriques, différentes méthodes sont nécessaires pour décrire l'association entre les variables.

Le reste de cette section décrit certaines des méthodes d'analyse des données de ce formulaire.

Alcool et apport de nicotine

Dans le cadre d'une étude de la façon boire et de fumer chez les femmes enceintes sur leurs enfants, les données ont été recueillies auprès de 452 mères au sujet de la relation entre leur consommation de nicotine pendant la grossesse et de leur consommation d'alcool avant leur grossesse a été reconnue. Le tableau de contingence ci-dessous décrit la relation entre ces deux variables catégorielles ordinales.

    Nicotine (milligrams/jour)
L'alcool (oz / jour) Aucun 1 à 15 Plus de 15
  Aucun 105 7 11
0.01 à 0.10 58 5 13
0.11 à 0.99 84 37 42
1.00 ou plus 57 16 17

Les variables ne peuvent pas être classées comme une réponse et des variables explicatives — les deux variables ont un statut similaire. Cependant, il est raisonnable de se demander si la consommation élevée d'alcool tend à être associée à un apport de nicotine élevé.
