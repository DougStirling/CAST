Spliting en groupes

Si les deux variables peuvent être traitées comme une réponse et une variable explicative, il est utile de diviser les données en «groupes» en utilisant la variable explicative, et comparer les distributions de la réponse dans les différents groupes . Ils sont également appelés les distributions conditionnelles de la réponse à chaque valeur de la variable explicative.

Même si les deux variables ne peuvent pas être classés dans une réponse et des variables explicatives, des méthodes similaires peuvent être utilisés. Si les variables sont appelés X et Y, nous pouvons soit

  • comparer les distributions de Y pour chaque valeur possible de X
  • comparer les distributions de X pour chaque valeur possible de Y

On les appelle les distributions conditionnelles de Y sachant X, et les distributions conditionnelles de X sachant Y, et les proportions au sein des groupes seraient utilisées pour faciliter les comparaisons.

Dans le cadre d'un tableau de contingence, les proportions conditionnelles sont trouvés en divisant chaque fréquence dans le tableau par sa ligne (ou colonne) au total. Cette échelle de chaque ligne (ou colonne) du tableau pour résumer à 1.0.

Sélectionnez Proportion dans le menu pop-up pour voir les distributions conditionnelles pour chaque groupe d'âge. En effet, cette échelle des fréquences dans chaque ligne du tableau de contingence à ajouter à 1,0. Cliquez sur la ligne pour les moins de 30 pour voir comment les proportions conditionnelles sont obtenues en divisant les fréquences conjointes par la fréquence marginale pour les moins de 30 ans.

Maintenant, choisissez dans le menu Rang pop-up sur le droit de voir les distributions conditionnelles pour chaque rang. Cliquez sur les colonnes de voir comment ces proportions conditionnelles sont obtenues à partir des fréquences communes.

Affichages graphiques des distributions conditionnelles

Les distributions conditionnelles peuvent être représentés graphiquement sur un diagramme à barres en 3 dimensions, mais un cluster à 2 dimensions d'affichage est généralement plus facile à interpréter. Notez cependant que plusieurs types d'affichages groupés peuvent être tirés — ils rendent plus facile de comparer les différents aspects de la distribution.

  • Les probabilités conditionnelles de X sachant Y ou Y étant donné X.
  • Regroupées par Y ou X.

Grade et l'âge

Le graphique à barres groupées ci-dessous montre d'abord les fréquences communes pour toutes les combinaisons d'âge et de rang.

Sélectionnez tout d'abord Rang dans le menu pop-up sous le pôle graphique à barres pour les barres en fonction du grade. Le nombre total d'enseignants est petit, donc il est difficile de campare les âges de moniteurs à ceux des autres grades. Sélectionnez Propn sein Rang dans le menu pop-up dans le haut pour afficher les distributions conditionnelles de l'âge dans le rang. Il efficace échelles bars chaque rang de donner le même total (1.0).

Il est maintenant facile de voir que la répartition par âge des professeurs adjoints et les instructeurs sont très similaires, mais les deux sont différents de ceux desprofesseurs agrégés et titulaires.

Sélectionnez la fréquence et l'âge dans les deux menus pour montrer les chiffres bruts, regroupés par âge. Sélectionnez Propn sein âge pour afficher les distributions conditionnelles des rangs du personnel qui sont dans chaque groupe d'âge.
