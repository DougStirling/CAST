Modèles quadratiques

Une tendance linéaire n'est pas approprié pour toutes les séries chronologiques. De nombreuses tendances ont courbure qui doit être décrit par un modèle plus complexe. Nous avons maintenant de décrire brièvement montage d'une tendance quadratique de la forme ...

tendance   =   b0  +  b1 temps  +  b2 temps2

Une courbe quadratique de cette forme a trois paramètres qui peuvent être ajustés pour améliorer l'ajustement du modèle. Nous avons de nouveau définir les résidus d'être les différences entre les valeurs réelles de séries chronologiques, y, et les prédictions du modèle,

ei   =   yi  −  tendancei

Les estimations les moins carrés des trois paramètres sont à nouveau les valeurs qui minimisent la somme des carrés des résidus,

La dette de carte de crédit

La tendance est clairement non linéaire, donc un modèle quadratique pourrait être appropriée. (Notez que nous avons à nouveau soustrait 1990 de l'année pour rendre les coefficients plus gérable.)
