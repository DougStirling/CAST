Ordre pour les lignes d'une table

Dans de nombreux tableaux, les lignes sont ordonnées alphabétiquement par leur nom consécutive, mais il est généralement préférable de les réorganiser dans un autre de façon significative.

Quelques données sur l'Afrique

Le tableau ci-dessous montre les trois colonnes de renseignements sur la santé de certains pays africains (surtout les données de 2003). Seuls les pays avec des populations plus de 10 millions ont été inclus pour garder la table à une taille gérable.

Les pays sont d'abord triés en ordre alphabétique. Cela permet de trouver rapidement les valeurs pour un pays particulier, mais contribue rarement vous de voir ce qui est associé à des différences entre les valeurs dans les colonnes.

Utilisez le menu pop-up pour réorganiser les pays du Nord au Sud. Cette commande permet de montrer s'il existe des motifs géographiques.

Essayez Suivant ordonnant le pays par leur PIB par habitant (avec le pays les plus riches au sommet). Cela pourrait montrer si la richesse des pays sont associés à des variables.

Enfin, essayez de commander les pays sur la base des variables qui sont affichées dans le tableau. Par exemple, l'ordre par les taux de tuberculose. Ne les pays avec des taux de tuberculose est élevée ont aussi élevée du VIH / sida les plus élevés? Moins d'infirmières?

Il n'ya pas de «bonne» façon de commander les lignes d'une grande table et le «meilleur» afin dépend des informations que vous souhaitez mettre en évidence. Cependant il existe des moyens habituellement mieux que l'ordre alphabétique.
