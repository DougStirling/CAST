Signal et le bruit

Tout affichage graphique ou tabulaire des données devrait être conçue pour mettre en évidence des caractéristiques importantes des données.Cette information utile à l'écran est appelé son signal. D'autres aspects de l'affichage qui ne contiennent pas des informations qui peuvent être utilement interprété sont appelés le bruit dans l'affichage.

Edward Tufte, dans un excellent livre sur la présentation des données (The Visual Display of Quantitative Information, 1983), distingue différents types de bruit dans les affichages.

Non-données sur le bruit
Il s'agit de graphiques inutiles et le quadrillage qui sont ajoutés aux écrans. Tufte recommande de réduire la quantité de «non-données d'encre» dans n'importe quel écran.
Le bruit des données
Le bruit des données sont des informations sur les données qui ne permet pas au lecteur de comprendre le «signal» dans les données. De nombreux rapports sont pleins de bruits de données — l'écrivain a passé du temps à recueillir des données et ne voulez pas manquer de tout ça, même si elle n'est pas pertinente!

Les deux types de bruit il est plus difficile à détecter le signal dans un écran, donc le bruit doit être évité.

Chiffres significatifs

Un type de données de bruit est très fréquent, mais facile à enlever. De nombreux tableaux contiennent des valeurs qui sont rapportés avec des chiffres plus importants que nécessaire. Habituellement, le modèle de valeurs dans un tableau peut être compris à partir de seulement leur premier 2 ou 3 chiffres — les chiffres restants sont le bruit des données.

(Si les données complètes peuvent être nécessaires par d'autres pour une analyse plus approfondie, les données complètes peuvent être inclus dans une annexe ou mis à disposition sur un site web, mais pas dans le corps d'un rapport.)

Couleur des voitures en Nouvelle-Zélande

Le tableau ci-dessous décrit les couleurs de toutes les voitures immatriculées en Nouvelle-Zélande en 2006.

Personne lecture de la table serait intéressé par les derniers chiffres-unes des valeurs. Utilisez le bouton «-» dans les fréquences de réduire le nombre de chiffres significatifs affichés.

Affichage des fréquences au millier près supprime le bruit des données de la table, mais conserve toutes les informations utiles.

De façon similaire, autour du proportions à 3 décimales — autres chiffres ne vous aident pas à comprendre les données.

Enfin cliquez sur la case Pourcentage d'afficher des pourcentages plutôt que des proportions. Cette multiplie simplement les proportions de 100, mais il supprime certaines des zéros et rend donc les valeurs ressortent mieux.
