Les données chronologiques

Beaucoup de jeux de données contiennent des mesures qui sont faites de façon séquentielle à des intervalles réguliers. Ces données sont appelées séries chronologiques.

  • Les climatologues analysent les documents historiques de la météo pour évaluer les preuves du réchauffement climatique.
  • Exportateurs regardez récente des taux de change pour aider à prévoir les mouvements futurs qui affecteront le prix de leurs produits sur les marchés étrangers.
  • Chaînes de détaillants de suivre l'évolution de la population dans les différentes régions pour aider à déterminer où les nouveaux magasins devraient être situés.
  • Les scientifiques de Santé d'examiner les séries chronologiques du nombre de cas de grippe pour aider à prévoir la demande pour les vaccins.
  • Les fabricants de recueillir régulièrement des données sur la qualité de leurs produits. Par exemple, la teneur en matières grasses du lait est susceptible d'être enregistrée quotidiennement par une usine d'embouteillage.

Les données chronologiques sont largement analysées dans le monde des affaires — des prévisions précises des taux de change, prix des actions, la demande pour les produits et les variables d'affaires d'autres peuvent avoir un effet majeur sur la rentabilité.

L'importance d'avoir comploté

Il est souvent difficile d'obtenir des informations utiles à partir des séries temporelles si elles sont présentées sous forme tabulaire. Comme avec d'autres structures de données, l'information dans une série chronologique est plus facile à comprendre à partir d'un affichage graphique.

Une parcelle de séries chronologiques est un type de tracé de points dans lequel les valeurs sont affichées comme des croix contre un axe vertical. L'axe horizontal se déploie la croix dans l'ordre du temps. (Il peut aussi être considéré comme un nuage dans lequel le «explicative» variable est le temps.) La croisements successifs sont souvent reliés par des lignes.
