La forme d'une relation est seulement connu dans les données

Les modèles que nous avons utilisés pour décrire la relation entre une réponse, Y et des variables explicatives, X, sont généralement que des approximations de la «vraie» relation. Par exemple, un nuage de points peut paraître linéaire, mais nous n'avons vraiment aucune information sur la forme de la relation au-delà de nos données.

En conséquence, les modèles que nous avons décrit peut être utilisé pour prédire Y à partir des valeurs de X qui sont dans la fourchette de valeurs x dans nos données, mais nous devrions être très prudents quant à l'aide d'un modèle équipé pour des prédictions dehors de cette plage. C'est ce qu'on appelle l'extrapolationet il peut être mal dans l'erreur.

Évitez d'utiliser un modèle pour prédire Y bien au-delà des données disponibles.

Quel modèle pour la courbure est la meilleure?

Nous avons décrit deux types de modèle non linéaire pour les données à courbure — modèles quadratiques et des modèles linéaires basés sur des variables transformées. Il ya beaucoup d'ensembles de données pour lesquels les deux types d'ajustement du modèle tout aussi bien.

Toutefois, les différents types de modèles peuvent donner des prévisions très différentes lorsqu'il est utilisé pour extrapoler à partir des données observées. Les données ne peuvent pas nous aider à décider ce qui est mieux.

Forbes des données

Un tel ensemble de données sont les données de Forbes, pour lequel nous espérons pour décrire la relation entre la pression barométrique, Y, et le point d'ébullition de l'eau, X.

Nous avons vu précédemment que le modèle quadratique monté ainsi, la courbe des carrés est moins

estimation de la pression   =   116.59  −  1.4165 x  +  0.004752 x2

Le logarithme de la pression est également à peu près linéairement liées à ébullition avec l'équation

estimation de ln (pression)   =  −0.9518  +  0.02052 x

Cette équation peut être réécrite sous la forme

estimation de la pression   =   exp (−0.9518  +  0.02052 x)

Les deux modèles (quadratique dans les mesures originales et linéaire dans la mesure transformé) à la fois ajuster les données aussi bien avec des parcelles résiduelles qui sont presque identiques. Les données ne donnent aucune indication sur lequel des modèles alternatifs convient le mieux.

Utilisez le menu déroulant pour afficher les deux lignes équipées (Modèle quadratique et Modèle log-linéaire). Observez qu'ils coïncident presque.

Sélectionnez l'option Les deux modèles, puis utilisez le curseur pour étendre les deux axes. Lorsque le point d'ébullition est inférieure à environ 150 degrés, soit plus de 250 degrés, les deux modèles donnent des prévisions très différentes de la pression.

En effet, le modèle quadratique (ce qui est vert) prédit que la pression va augmenter lorsque le point d'ébullition est inférieure à 150 degrés, ce qui n'est pas conforme à la physique! Pour ces données, le modèle log-linéaire est susceptible de donner de meilleures prévisions à bas points d'ébullition.
