Ajout d'un terme quadratique

Une solution alternative au problème de la courbure est d'étendre le modèle linéaire simple avec l'ajout d'un terme quadratique,

y  =  b0  +  b1 x  +  b2 x2

Valeurs ajustées et les résidus sont définis (et interprété) d'une manière similaire à ceux d'un modèle linéaire,

  =  b0  +  b1 xi  +  b1 xi2
ei  =  yi −

Comme dans un modèle linéaire, les résidus du modèle quadratique sont les distances verticales entre les croix dans un nuage de points et la courbe. Nous avons de nouveau l'utilisation des moindres carrés pour estimer les paramètres inconnus — choisir des valeurs des trois paramètres de minimiser la somme des carrés des résidus,

Le diagramme de dispersion ci-dessous montre un ensemble de données avec une relation non linéaire.

Faites glisser les trois flèches rouges pour ajuster la position de la courbe quadratique. Observez que ...

  • Le terme constant est la valeur y correspondant à x = 0.
  • Le coefficient de x est la pente de la courbe en x = 0.
  • Le troisième coefficient décrit la courbure, elle est petite quand la courbe est presque linéaire.

Cliquez sur le case Résidus afficher. Les résidus sont affichées avec des lignes verticales bleues sur le diagramme de dispersion. Ajuster les coefficients en utilisant les flèches rouges pour faire des résidus aussi petit que possible, puis cliquez sur le bouton de moindres carrés à minimiser la somme résiduelle des carrés.

Si une relation n'est pas linéaire, les résidus d'un modèle quadratique sont susceptibles d'être plus petits que ceux d'un modèle linéaire. Cela suggère aussi que les erreurs sont susceptibles d'être moindre si le modèle est utilisé pour prédire les valeurs de réponse avenir.

Forbes des données

Nous avons essayé auparavant d'utiliser un modèle linéaire pour expliquer la relation entre la pression barométrique et la température d'ébullition de l'eau. Le diagramme de dispersion des résidus sur la droite ci-dessous montre les problèmes avec ce modèle linéaire.

Tout d'abord, cliquez sur la case Supprimer pour supprimer la valeur aberrante aberrantes que nous avons identifiés plus tôt. Notez encore la forte impression de courbure dans le tracé des valeurs résiduelles sur la droite.

Ensuite, choisissez Ajustement quadratique option dans le menu déroulant. La courbe de moins carrés quadratique est affiché sur la gauche, et le tracé des valeurs résiduelles sur la droite affiche les résidus du modèle quadratique. Notez que ...

  • L'ampleur des résidus est plus petite, de sorte que le modèle quadratique prédit une pression plus grande précision.
  • Il n'ya pas de motif dans les résidus qui pourraient jeter le doute sur le modèle quadratique.
