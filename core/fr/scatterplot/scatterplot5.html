Time-commande de données univariées

Certains ensembles de données sont apparemment univariée, mais les mesures sont effectuées de façon séquentielle dans le temps. Par exemple, beaucoup de mesures économiques et démographiques sont enregistrées mensuellement ou annuellement. Un ensemble de données de ce formulaire est appelé une série chronologique.

L'heure à laquelle chaque mesure a été rendue peut être traitée comme une variable supplémentaire numériques, et la mesure peut alors être relevée en temps. Ce type de nuage est souvent appelé un complot de séries chronologiques.

Les séries chronologiques peuvent être visualisées et analysées avec beaucoup de techniques qui sont utilisées pour les données bidimensionnelles

Autres méthodes d'analyse qui sont spécifiques à des données de séries chronologiques seront présentés dans un chapitre ultérieur.
