Les diagrammes de dispersion sont nécessaires pour afficher les relations

La relation entre deux variables ne peut pas être déterminée à partir de l'examen de ces deux variables dans l'isolement.

Le schéma ci-dessous montre les distributions univariées des deux variables X et Y comme point Jittered parcelles par rapport aux axes.

Pouvez-vous deviner la relation entre X et Y pour les deux parcelles de points Jittered qui sont présentés par rapport aux axes?

Utilisez le menu pop-up pour l'affichage des nuages ??de points différents qui sont compatibles avec les parcelles univariée points Jittered. Nous verrons plus tard que ces nuages ??de différentes impliquent des types très différents de la relation entre les variables. Il est donc important d'utiliser un nuage de points pour déterminer le type de relation.
