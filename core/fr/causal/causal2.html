Association

Comme indiqué dans la page précédente, les chercheurs sont généralement intéressés par les relations entre les variables. Lorsque deux variables sont liées, nous disons qu'il ya association entre eux.

Par exemple, considérons la hauteur, X, et le poids, Y, d'un échantillon d'enfants scolarisés. Les enfants ont tendance à être grands lourdes, donc des valeurs élevées deX sont associés à des valeurs élevées de Y. Le coefficient de corrélation décrit la quantité d'association linéaire entre deux variables numériques tels.

Les relations causales

Dans certains ensembles de données, il est possible de conclure qu'une variable a une influence directe sur l'autre. C'est ce qu'on appelle une relation causale.

Par exemple, ...

  • Un chercheur effectue une réaction chimique avec différentes concentrations d'une emzyme et les enregistrements de la pureté du produit final à chaque concentration. Le emzyme pourraient influencer la pureté, mais la pureté de la sortie ne peut pas influencer la quantité de emzyme utilisé. La relation est donc causal.
  • Un fermier tente deux différents traitements vermifuges sur les troupeaux de moutons qui sont par ailleurs identiques. Toute différence entre les proportions de moutons infectés après les deux traitements suggère que le traitement affecte causalement infection.

Si deux variables sont causalement liés, il est possible de conclure que les changements de la variable explicative, X, aura un impact direct sur Y.

Non-causale des relations

Pas toutes les relations sont causales. En non-causale des relations, la relation qui se manifeste entre les deux variables n'est pas complètement le résultat d'une variable affectant directement l'autre. Dans le cas le plus extrême, ...

Deux variables peuvent être liées les unes aux autres, sans variables affectant directement les valeurs de l'autre.
