Tableau de fréquences

Les diagrammes à barres pour les données discrètes sont basées sur les fréquences des différentes valeurs — c'est à dire le nombre de fois que chaque valeur apparaît dans l'ensemble de données.

Dans les ensembles de données avec un petit nombre de chefs d'accusation possibles (disons 20 ou moins), un tableau de fréquence est un résumé utile dans son propre droit. Contrairement à des tableaux de fréquence pour les données continues, aucun groupement est impliqué afin qu'aucune information n'est perdue.
