Des données discrètes et continues

Dans cette section, nous distinguons deux types de données numériques.

Les données discrètes
Lorsque les valeurs dans le lot sont des nombres entiers (chiffres), l'ensemble des données est appelé discret. Exemples de mesures discrètes sont les suivants:
  • le nombre d'admissions dans un accident d'un hôpital et l'unité d'urgence de chaque jour sur une période de deux mois,
  • le nombre de personnes dans chaque ménage dans une enquête auprès de 10.000 ménages,
  • le nombre d'œufs cassés dans chacune des 200 boîtes de 1 douzaine d'œufs arrivant à un supermarché,
  • le nombre de numéros d'un magazine en particulier acheté au cours des 12 mois précédents par chaque répondant à un sondage téléphonique.
Les données continues
Lorsque les données ne sont pas contraints d'être des nombres entiers, l'ensemble des données est dite continue. Les exemples sont:
  • les températures maximales de chaque jour en Janvier dans votre ville,
  • la concentration d'un polluant dans chacun des 20 échantillons ou l'eau des rivières qui ont été recueillies sur une période de 1 semaine,
  • l'apport calorique quotidien de chaque enfant dans une enquête sur la santé,
  • le taux de change quotidiens entre le dollar néo-zélandais et le dollar australien au cours du mois passé.
     

Parcelles Point pour compte

Dot parcelles peut être utilisé pour afficher des données de comptage. Cependant quand les comptages sont toutes de petite taille, de nombreuses valeurs apparaissent habituellement plusieurs fois. Base parcelles points sont donc trompeurs, car les valeurs répétées sont superposées et apparaissent comme une simple croix.

Cependant empilés et des parcelles de points Jittered peuvent encore être utilisés. Si les chiffres sont tout petits, aucune information doivent être perdue par l'empilement puisqu'il peut y avoir une colonne de croix pour chaque valeur distincte.
