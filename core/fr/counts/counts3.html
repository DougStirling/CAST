Afficher les petits chiffres

Lorsque la plage de valeurs dans un ensemble discret de données est petite, un histogramme peut être établi avec une largeur de classe sans paraître trop irréguliers. Ces classes sont centrées sur les valeurs possibles dans l'ensemble de données (soit 1, 2, 3, etc.)

Un tel histogramme peut être améliorée en réduisant les rectangles de sorte qu'ils ne se touchent pas, car cela souligne la nature discrète des données. L'affichage résultant est appelé un graphique à barres des données.

Pour des données discrètes, les graphiques à barres sont préférables aux histogrammes, autant que cela ne résulte pas dans des classes trop nombreuses.
