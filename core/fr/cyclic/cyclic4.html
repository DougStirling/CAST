Dangers de sur-interprétation

Très forte autocorrélation dans une série temporelle peut entraîner des tendances apparentes, même quand il n'ya pas de véritable tendance sous-jacente de la série. Un exemple important de cela est le modèle «marche aléatoire» pour les données de prix d'actions. Autocorrélation et tendance sont particulièrement difficiles à distinguer dans les séries chronologiques courtes. Vous devriez donc se méfier de l'extrapolation d'une tendance dans le futur s'il ya autocorrélation élevée.

Le diagramme ci-dessous montre une série temporelle qui a été artificiellement généré sans tendance et avec les valeurs successives qui ne sont pas autocorrélés.

Cliquez Nouvel échantillon quelques fois pour observer les types de modèle qui pourrait être observé avec une série chronologique non corrélées.

Faites glisser le curseur pour augmenter le coefficient d'autocorrélation à 0.95 puis prendre quelques échantillons de plus d'observer les types de modèle qui peuvent être trouvés dans les données autocorrélées. Notez que certaines séries semblent montrer la tendance, même si le sous-tend aucune des données.

(Essayez de faire le coefficient d'autocorrélation négative. Avec autocorrélation négative, les valeurs positives ont tendance à être suivies par celles qui sont négatives et vice versa. Ce type de série est rarement dans la pratique, bien que cela peut arriver quand il ya un mécanisme de rétroaction exagérée entraînant l'ajustement automatique vers un niveau d'équilibre.)
