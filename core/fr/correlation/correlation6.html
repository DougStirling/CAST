Toujours regarder une première nuage

Bien que le coefficient de corrélation est une bonne description de la force de nombreuses relations, il ne décrit pas adéquatement les autres.

Un nuage de points doivent toujours être examinés pour aider à évaluer s'il existe des fonctions dans les données que le coefficient de corrélation ne peut pas décrire.

Anscomb de données

Le diagramme ci-dessous montre un diagramme de dispersion et de plusieurs statistiques sommaires numériques, y compris le coefficient de corrélation.

Le menu pop-up vous permet de sélectionner l'un des ensembles de données de quatre. Tous les quatre ensembles de données ont les mêmes statistiques sommaires, mais le nuage de points montre que les relations entre X et Y sont très différentes.

Modéré relation linéaire
Il ya une assez forte relation linéaire positive entre X et Y et les statistiques sommaires décrire convenablement cet ensemble de données.
Forte relation non linéaire
Tous les points se situent exactement sur une courbe sorte que la relation est plus forte que suggéré par le coefficient de corrélation.
Forte linéaire & aberrantes
Dans cet ensemble de données, toutes les croix sont sur une ligne droite, sauf pour une valeur aberrante unique. Est-ce une transcription ou une erreur de mesure?
Seul un X est différent
X = 8 pour tous les points sauf un où X = 19. Ce X pourrait être une valeur aberrante, mais même si elle est une mesure réelle, le point a une influence extrême sur le coefficient de corrélation. Changer sa valeur y à 7 permettrait de réduire le coefficient de corrélation de zéro. La force apparente de la relation est déterminée par la valeur y de ce point, il est donc appelée très influents.

Le message important est:

Ne vous contentez pas de calculer des statistiques sommaires, regarder un nuage de points de données aussi.
