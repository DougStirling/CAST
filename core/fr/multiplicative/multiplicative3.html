Obtenir des prévisions

Dans cette page, nous considérons les détails des prévisions avec des modèles multiplicatifs de la forme

Données   =   (Effet saisonnier)   ×   Tendance   ×   Cyclique   ×   Résiduelle

Vous avez appris que ce modèle est équivalent à un modèle additif pour les données du journal.

log(Données)   =  Saisonnier*   +   Tendance*   +  Cyclique*   +  Résiduelle*

où les nouveaux composants sont les logarithmes de ceux de l'équation précédente.

Le modèle est donc équipé d'abord de prendre les journaux des valeurs des données originales et montage d'un modèle additif de la manière habituelle. Toutefois, ce modèle donne des prévisions de log(données).

prévision de log(Données)   =   (Prévision de Saisonnier*)  +  (Prévision de Tendance*)  +  (Prévision de Cyclique*)

Pour obtenir des prévisions à l'échelle d'origine, une transformation inverse doit être appliquée. Si les logarithmes de base 10 ont été utilisés (comme dans les schémas de CAST), la transformation inverse est appropriée

prévision des Données   =   10 prévision de log(Données)
 

Si les logarithmes naturels sont utilisés, la fonction exponentielle doit être utilisée pour retourner la prévision basée sur les données du journal d'une prévision dans les unités d'origine. (Les deux types de résultat dans les prévisions de logarithmes identiques.)

Cochez les cases Saisonniers, Tendance et Cyclique pour afficher la somme de ces composants. Les composants sont également prévues trois années dans le futur dans le diagramme. Cliquez sur n'importe quelle prévision pour l'afficher numériquement, transformé à nouveau à l'échelle originale.
