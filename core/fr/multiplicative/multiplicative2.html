Différence entre additif et modèles multiplicatifs

Modèles multiplicatifs pour les séries chronologiques sont très largement utilisés. Il est donc intéressant veillant à ce que vous pouvez reconnaître les caractéristiques qui suggèrent un modèle multiplicatif.

Modèles additif et multiplicatif ne peuvent être facilement distingués quand il ya tendance notable dans les séries chronologiques.

Modèle additif
La variation saisonnière, cyclique et résiduel autour de la ligne de tendance sont de la même taille appoximately si la valeur de tendance est faible ou élevé.
Modèle multiplicatif
La variation saisonnière, cyclique et résiduel autour de la ligne de tendance sont plus importants lorsque la tendance est à des valeurs élevées que lorsque la tendance est faible

Modèle additif pour les visiteurs de Nouvelle-Zélande

Le graphique chronologique ci-dessous décrit les arrivées de visiteurs en Nouvelle-Zélande (en milliers).

Cliquez sur Tendance cases puis Saisonnière à ajuster un modèle additif à ces données. Observez que les fluctuations saisonnières dans les données sont plus importantes ces dernières années, mais le «meilleur» modèle additif ne peut pas correspondre à cela.

En conséquence, le modèle additif surestime les fluctuations saisonnières dans les années 1980 mais les sous-estimations dans les années 1990. Cliquez sur les valeurs dans la dernière année afin de vérifier que le modèle sous-estime le nombre de touristes en été en Nouvelle-Zélande (Q4 et Q1) et le nombre de touristes sur-estimations de l'hiver (Q2 et Q3).

Prévisions trimestrielles basées sur le modèle additif sera donc similaire du nombre de touristes sous-estimer l'été en Nouvelle-Zélande et les surestimer l'hiver.

Modèle multiplicatif pour les visiteurs de Nouvelle-Zélande

Le graphique chronologique ci-dessous montre le nombre de visiteurs sur une échelle logarithmique.

Le modèle multiplicatif est équivalent à un modèle additif montage aux données du journal. Cliquez sur Tendance cases puis Saisonniers pour afficher ces composants. Sur l'échelle logarithmique, l'effet saisonnier est le même pendant toute la période de temps.

Maintenant, utilisez le curseur pour ajuster l'échelle revenir à un tracé des données d'origine. Le modèle multiplicatif implique une plus faible variation saisonnière où les arrivées de visiteurs sont faibles, et la variation saisonnière plus élevée lorsque le nombre de visiteurs est élevé. Cela correspond à la tendance dans les données de puits.

Ce modèle multiplicatif est susceptible de donner de meilleures prévisions que le modèle additif — il n'ya pas de tendance à systématiquement quarts surestimer ou sous-estimer dans les années les plus récentes.
