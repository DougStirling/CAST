Les variables catégorielles et les groupes

Une variable catégorique peut être utilisé pour diviser les individus dans un ensemble de données en groupes. Par exemple, nous pourrions enregistrer le poids et la couleur de la coquille (blanc ou brun) d'œufs de poules 50. Nous pourrions utiliser la variable «couleur de la coquille» pour diviser les œufs dans les œufs blancs et les œufs bruns et séparément la liste des poids des deux groupes.

Inversement, si les données ont été collectés séparément à partir de différents groupes de personnes, les ensembles de données résultant pourrait être combinée avec une variable catégorique la distinction entre les groupes. Par exemple, une expérimentation pourrait être menée pour comparer les durées de vie de trois marques de ampoule électrique. Les trois ensembles de vies pourraient être combinées en un seul ensemble de données avec une variable numérique (la vie) et une variable catégorielle à distinguer les trois marques. Ce type de jeu de données se pose souvent à partir d'expériences.

Une variable catégorique et les groupes sont souvent deux façons de représenter les mêmes données.

Enquête sur le riz

Dans le cadre d'une enquête auprès des producteurs de riz au Sri Lanka, 36 agriculteurs ont été sélectionnés au hasard dans 4 villages. Le rendement du riz (en tonnes par hectare) a été déterminée à partir de chaque agriculteur.

Ces données sont naturellement présenté comme une liste séparée des rendements pour chaque village. Cliquez sur valeurs sur la gauche pour voir comment ils sont représentés en utilisant une catégoriques «Village» variable dans une matrice de données.

Jours de congé de maladie

Le directeur d'une usine a été préoccupé par le nombre de jours que les travailleurs ont pris congé chaque année pour cause de maladie. Les dossiers du personnel pour les 60 travailleurs ont été examinés, et leurs jours de maladie ont été enregistrés avec leur groupe d'âge, les vieux (défini comme 40 ans ou plus) ou jeunes (moins de 40).

Ces données sont naturellement enregistrées dans une matrice de données avec des colonnes pour les jours de congé et pour le groupe d'âge, mais l'âge des travailleurs peut être utilisé pour diviser l'autre variable en groupes. Cliquez sur la rangée du haut du tableau de données et faites glisser vers le bas pour voir comment les valeurs numériques sont divisés en groupes.
