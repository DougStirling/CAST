L'analyse exploratoire des données

Les premiers chapitres de ce livre électronique décrire les méthodes statistiques pour étudier et résumer un ensemble de données. Des méthodes appropriées dépendent de la structure de l'ensemble des données — le nombre et les types de ses variables.

Une variable numérique
Le chapitre 2 décrit résumés graphiques et numériques d'une seule variable numérique.
Deux ou plusieurs variables numériques
Le chapitre 3 examine la relation entre deux ou plusieurs variables numériques.
Un ordonnée variable numérique
Dans le chapitre 4, nous examinons des méthodes statistiques pour données ordonnées (série chronologique).
Les variables catégorielles
Dans le chapitre 5, les méthodes graphiques et numériques sont présentés pour les variables catégorielles.
De nombreuses variables
Le chapitre 6 est un bref chapitre contenant quelques méthodes pour les ensembles de données avec 3 ou plusieurs variables.

Inférence

Pour bien comprendre l'information qui est contenue dans la plupart des ensembles de données, nous devons tenir compte du caractère aléatoire — si nous avons recueilli les données à nouveau en répétant une expérience ou la collecte de données provenant de différentes personnes, les valeurs seraient souvent différents. Les méthodes statistiques pertinentes sont collectivement appelés inférence.

Encore une fois, les détails de l'analyse statistique dépendra principalement de la structure de l'ensemble des données — le nombre et les types de ses variables.

Une variable
Les chapitres 9 et 10 de développer la méthodologie pour répondre aux questions sur une seule variable numérique ou catégorielle (intervalles de confiance et tests d'hypothèses).
Deux ou plusieurs groupes
Chapitre 11 des intervalles de confiance actuelle et des tests d'hypothèse pour comparer deux ou plusieurs groupes.
Deux variables numériques
Chapitre 12 modèles de la relation entre deux variables numériques et donne confiance et tests d'hypothèses concernant ces modèles.
Deux variables catégorielles
Le chapitre 13 décrit la méthodologie pour comparer des groupes de données catégoriques ou d'examiner la relation entre deux variables catégorielles (tableaux de contingence).

La collecte des données

Les statisticiens devraient être impliqués avant que des données sont collectées. Principes statistiques peuvent être appliquées à la collecte des données qui garantissent que les données obtenues peuvent être analysées de manière significative. Les chapitres 7 et 8 d'explorer l'idée d'un échantillonnage aléatoire et de décrire certains des principes qui devraient être suivies dans la collecte de données.
