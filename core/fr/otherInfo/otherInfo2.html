Problèmes de navigateur?

Si vous pouvez obtenir dans un CAST e-book, mais ne pouvez pas voir les diagrammes interactifs, les raisons les plus probables sont que vous ne disposez pas d'une version récente de Java installée sur votre ordinateur, ou que Java n'a pas été activé dans votre navigateur.

Certaines fonctionnalités de CAST exigent également un navigateur relativement récent. Par exemple, certains navigateurs plus anciens ne présentent pas «σ» comme une lettre «sigma» grec.

La page sur exigences du navigateur donne plus de détails.
