Les navigateurs qui supportent CAST

Il n'y a pas de problèmes connus lors de l'utilisation CAST avec les versions récentes de ...

  • Internet Explorer
  • Safari
  • Firefox

Notez cependant que les diagrammes interactifs CAST ne fonctionnent plus avec Google Chrome, de sorte que le navigateur doit être évité. Si vous avez eu des problèmes avec les onglets de la page à partir de laquelle vous avez commencé cet e-book ou si votre navigateur ne σ affiche pas comme une lettre grecque 'sigma', alors vous devriez mettre à jour vers une version plus récente de votre navigateur.

Support de Java dans votre navigateur

Les diagrammes interactifs CAST sont tous programmés dans le langage de programmation Java et tous les navigateurs récents peuvent exécuter ces programmes Java (applets). Cependant Java ne sont pas automatiquement pris en charge par les systèmes d'exploitation récents et la dernière version doit être installé séparément des site Web principal de Java d'Oracle.

Quelques problèmes de sécurité avec Java ont également causé des systèmes d'exploitation et navigateurs pour empêcher les applets Java de fonctionner automatiquement, donc vous devrez peut-être modifier certains paramètres du système d'exploitation et navigateur avant de pouvoir fonctionner. Des conseils spécifiques pour votre ordinateur est fourni lorsqu'un CAST e-book est démarré.
