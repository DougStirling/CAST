Recent information about installing and running CAST

 

 
Java
The interactive diagrams in CAST are implemented with the Java language and environment and this must be installed on your computer before they can be seen. When you start any CAST e-book, the installed version of Java is checked. CAST needs a fairly recent version of Java and you must install Java from the Oracle's Java web site if the initial check reports that it is needed.
Java security settings
In recent operating systems, the default security settings for Java have been tightened and these often need to be modified. Specific advice about the settings for your browser and operating system are shown when you start any CAST e-book. Unfortunately the latest release of Google Chrome no longer supports Java, so the interactive diagrams in CAST no longer work with it.
Browsers
CAST needs a fairly recent web browser, so it is recommended that you upgrade to the latest release of your browser. CAST works with the latest releases of Firefox, Safari and Internet Explorer, but not Chrome since it no longer supports Java.

We have found that Firefox and Safari are the most stable browsers so we recommend them.
