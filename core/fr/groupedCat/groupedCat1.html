Les données catégoriques de plusieurs groupes

Des informations utiles peuvent parfois être obtenus par l'examen d'une seule distribution catégorielle avec des graphiques à barres ou camemberts. Cependant des questions plus intéressantes peuvent généralement être demandé de données quand ils sont obtenus à partir de plusieurs groupes.

  • Les types de cancer subis par les différentes races de même?
  • Les étudiants matures obtenir les grades similaires (A, B, C ou échec) à de jeunes étudiants dans une université?
  • Quelle est l'efficacité des pesticides naturels dans la réduction de la chance d'obtenir des plantes de choux infestés de chenilles?

Toutes les questions impliquent des comparaisons d'une distribution catégorielle (type de cancer, le grade, l'infestation, ...) pour différents groupes (les courses, le type d'étudiants, de pesticides, ...).

Tableaux de contingence

En supposant à nouveau que l'ordre des valeurs est sans importance, les données catégoriques dans chaque groupe peut être exprimé comme un tableau de fréquences. La combinaison de ces tableaux de fréquence dans un tableau rectangulaire unique donne un tableau de contingence.

Enquête sur le riz

Dans le cadre d'une enquête auprès des producteurs de riz au Sri Lanka, 36 agriculteurs ont été sélectionnés au hasard dans 4 villages. Chaque agriculteur échantillonnés a été interrogé sur la variété de riz qu'il a utilisé et les variétés ont été classées en «Vieux», «Traditionnel» ou «Nouveau». Les 36 valeurs résultantes catégoriques sont regroupés par village sur la gauche du schéma ci-dessous.

Cliquez sur toutes les valeurs de Sabey pour construire les fréquences dans la première colonne du tableau de contingence. Répéter avec les valeurs des autres villages de compléter le tableau.

Les données ne peuvent être présentées comme des listes distinctes de valeurs de chaque groupe. Les groupes peuvent être définis de façon équivalente par une variable catégorielle dans la matrice de données originale. Chaque «individuelles» contribue à nouveau un compte de 1 à une seule cellule du tableau de contingence.
