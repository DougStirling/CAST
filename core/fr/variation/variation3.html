Différences intentionnelle

Dans les expériences, les différents «individus» sont donnés différents traitements expérimentaux avec l'intention de comparer ces traitements. Par exemple:

  • Certains patients souffrant d'hypertension artérielle sont donnés un médicament nouvellement mis au point pour améliorer leur condition et d'autres sont donné un médicament standard.
  • Répétez différents passages d'une réaction chimique se fait à des températures différentes.
  • Une entreprise envoie des lettres aux sommes dues aux clients. Certaines lettres menacent action judiciaire alors que d'autres simplement à rappeler aux clients de leur dette.

L'intention de ces expériences est de déterminer si ces différences dans le traitement expérimental provoquent des différences dans les mesures.

Signal et le bruit dans les données expérimentales

Dans les ensembles de données expérimentales, la variabilité causée par différentes conditions expérimentales est le signal dans les données puisque l'intention de l'expérience est de déterminer l'effet de ces différences.

Cependant, le signal dans les données est généralement masquée par la variabilité naturelle dans les données — le bruit dans l'ensemble de données.

Un rôle majeur de la statistique est d'estimer le signal à partir de données qui contiennent aussi du bruit.

Effet de l'engrais sur les plantes de tomate

Dans le diagramme ci-dessous, 20 plants de tomates sont cultivées comme décrit précédemment, mais 10 d'entre eux sont donnés ajout d'engrais. En pratique, cette expérience sera menée pour déterminer l'effet de l'engrais sur le rendement, mais dans cet exemple, on sait que l'engrais augmente le rendement de 0.5 kg par plante.

Observez que la variation naturelle des rendements obscurcit l'effet de l'engrais. A partir de ces données, vous pouvez même ne pas être certain que l'engrais a un effet quelconque. Cliquez sur Un autre 20 usines de répéter l'expérience avec d'autres plants de tomates.

Cliquez sur la case Vary effet. Le curseur vous permet de régler l'effet réel de l'engrais. Observez que lorsque l'effet des engrais est grand, il peut être assez clair que l'engrais ne améliorer le rendement - le signal est importante par rapport au bruit. Toutefois, lorsque l'effet des engrais est faible, le signal est tellement petite que les masques de bruit qu'elle.
