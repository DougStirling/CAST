Un modèle linéaire pour les proportions?

Lorsque nous avons modélisé la manière dont une variable explicative numérique effectué une variable de réponse numérique, une équation linéaire a été utilisée,

  =  b0 + b1 x

Lorsque la variable de réponse est catégorique, il est tentant d'essayer une équation linéaire analogue à expliquer comment la proportion dans une catégorie de réponse est affecté par la variable explicative,

prédiction de proportion,   

Malheureusement, ...

... une équation linéaire n'est pas approprié pour une proportion car il pourrait entraîner dans des proportions prédit supérieur à 1.0 ou inférieur à 0.0.

Modèles non linéaires

Pour modéliser la manière dont une proportion dépend d'une variable explicative numérique, X, une équation doit donner des valeurs comprises entre 0 et 1 pour toutes les valeurs possibles de X. Cela signifie que l'équation doit être non linéaire dans X.

Les menstruations et l'âge

Une ligne droite a été établi sur les données ci-dessous pour les menstruations modèle de la façon dont le menstruée proportion pourrait dépendre de l'âge.

Faites glisser la ligne rouge verticale sur l'axe pour obtenir le pourcentage prédit menstruations à un âge différent.

Le modèle linéaire est un raisonnablement proche ajustement aux données entre les âges de 111/2 et 141/2. De la pente de la droite (environ 0.25), nous pouvons dire qu'environ un quart des filles commencent menstruations chaque année.

Cependant, le modèle linéaire prédit que plus de 100% des filles auront eu leurs premières règles à un âge supérieur à 15, et une proportion négative menstruations à un âge inférieur à 11.
