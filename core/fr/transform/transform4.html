Une famille de transformations non linéaires

Beaucoup de jeux de données qui se posent dans la pratique portent sur des quantités qui ont distributions skew. Une transformation logarithmique peut enlever asymétrie, mais parfois, une classe de transformations plus flexible est nécessaire.

Un groupe de transformations appelées les transformations de puissance est souvent utilisé. Une transformation de puissance soulève chaque valeur dans l'ensemble de données pour une puissance p, où p est habituellement une certaine constante entre -2 et 2. Bien que ces valeurs de p sont les plus faciles à interpréter, les valeurs intermédiaires peuvent également être utilisés.

(Notez que toute valeur x, élevé à la puissance 0 est égal à 1.0, de sorte qu'il semble de prime abord que p = 0 ne donne pas une transformation utile. Cependant, quand p devient proche de 0, l'effet est similaire à une transformation logarithmique.)

Populations africaines

Le tracé de points ci-dessous décrit les 1987 populations de tous les pays de l'Afrique. Glisser sur la croix pour enquêter sur les valeurs aberrantes. Nous allons utiliser la puissance des transformations des populations d'étaler les pays à faible population et de réduire l'impact visuel de la valeur aberrante.

Faites glisser la ligne rouge verticale sur l'axe vers la droite. Cela réduit la puissance utilisée dans la transformation de sa valeur initiale de p = 1. Après avoir cliqué sur l'axe, les touches fléchées de votre clavier peut également être utilisé pour un réglage plus fin de p. Note that ...

  • p = 1 correspond à aucune transformation.
  • Diminuer p propage les valeurs inférieures et compresse les valeurs plus élevées.
  • L'augmentation de p se répand les valeurs supérieures et comprime les valeurs inférieures.
  • Lorsque p < 0, le pays le moins peuplé (Seychelles) apparaît toujours au bas de la parcelle. Ceci résulte du changement de signe.
  • Comme la valeur de p est proche de 0, toutes les valeurs se rapprochent de 1.0 (ou -1.0 lorsque p est négatif). L'ampleur de la transformation log (correspondant à p = 0) est très différente de celles pour p = 0.01 et p = -0.01, mais son effet sur la forme de la distribution est intermédiaire.

Nous avons étiqueté l'axe avec les valeurs transformées pour aider à expliquer la mécanique de transformations de puissance. En pratique, il est préférable d'étiqueter l'axe avec les mesures d'origine. Sélectionner les valeurs des options Raw dans le menu pop-up. Les étiquettes sur l'axe deviennent les populations des pays.

Réglez la puissance encore et observer l'effet sur les étiquettes. Notez la transition en douceur entre la transformation logarithmique et les pouvoirs de chaque côté.

Pour ces données, une transformation logarithmique est de nouveau proche mieux pour éliminer l'asymétrie. Remarquez que le «aberrantes», au Nigeria, ne se tient plus du reste des pays — il est compatible avec la distribution des populations dans le reste de l'Afrique.
