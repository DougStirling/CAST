Positionnement de la ligne pour donner les petits résidus

Les résidus d'un modèle linéaire sont les distances verticales de la croix à la ligne sur un nuage. Ils indiquent à quel point les prédictions de la ligne (les valeurs ajustées) correspondent aux réponses réelles dans les données.

Ainsi le positionnement de la ligne pour faire le petit résidus améliore l'ajustement du modèle aux données. Les paramètres b0 et b1 sur le modèle linéaire devrait être fixé pour rendre les résidus aussi petits que possible.

Dans le diagramme ci-dessous, les résidus pour toutes les observations sont affichées sous forme de lignes verticales bleues.

Faites glisser les flèches rouges sur le schéma d'ajuster l'interception et la pente de la ligne pour faire les résidus de petite taille.
