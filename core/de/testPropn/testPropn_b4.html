Wie man den p-Wert für einen einseitigen Test findet

Der Hypothesen-Test bei den abgeschreckten Walen benutzte eine Zufallsstichprobe der Größe n einer Population mit der Wahrscheinlichkeit des Erfolges π (sofortiges Verlassen der Wale). Die gesammelten Daten waren x Erfolge und wir testeten die Hypothese…

H0 :   π  =  π0
HA :   π  >  π0

…wo π0 die gefragte Konstante war (z.B. 0,4 in diesem Beispiel). Die folgenden Schritte wurden unternommen, um den p-Wert für den Test zu erhalten.

  1. Der Stichprobenanteil der Erfolge (p) wurde als die informativste Parameterstatistik für π identifiziert.
  2. Die Anzahl der Erfolge x = np hat eine normale Binomialverteilung ohne unbekannte Parameter, wenn H0 bestehen bleibt, deshalb ist es eine bessere Teststatistik.
  3. Der p-Wert ist die Summe von Restwahrscheinlichkeiten in dieser Binomialverteilung.

Das Ratten-Beispiel war ähnlich, aber die Alternativhypothese beinhaltete hohe π-Werte und der p-Wert wurde gefunden, indem man obere Restwahrscheinlichkeiten abzählte.

Wie man den p-Wert für zweiseitige Tests findet

Welche der Restwahrscheinlichkeiten am passendsten ist, hängt von der Alternativhypothese ab. Wenn die Alternativhypothese entweder hohe oder niedrige Werte von x zulässt, nennt man den Test zweiseitig.

H0 :   π  =  π0
HA :   π  ≠  π0

Der p-Wert ist dann das Doppelte der kleinen Restwahrscheinlichkeit, da die Werte von x in beiden Resten der Binomialverteilung den Beweis für HA liefern würden.

Blutgruppen in Somalia

In einer Studie über "Sab Bondsmen", einer Bevölkerungs-Untergruppe in Nord-Somalia wurden Blut-Test an einer Stichprobe von 54 durchgeführt, um herauszufinden, ob diese sich genetisch von der Hauptbevölkerung der "Free-born Noble Somali" unterscheiden.

Es ist bekannt, dass ein Anteil von 0,574 der Free-born Noble Somali die Blutgruppe 0 haben. (In einer Stichprobe von 1000 hatten aktuell 574 die Blutgruppe 0, aber diese Stichprobengröße war groß genung, um eine ziemlich genaue Schätzung zu liefern.) Gibt es einen Beweis, dass der Stichprobenanteil mit Blutgruppe 0 bei den Sab Bondsmen, 26 aus 54, nicht aus einer Population mit π  = 0,574 kommt? Dies kann in folgender Hypothese ausgedrückt werden 

H0 :   π  =  0.574
HA :   π  ≠  0.574

Wir würden erwarten, dass (0,574 x 54) = 31 der Sab Bondsmen die Blutgruppe 0 haben. Eine Stichprobenanzahl, die entweder viel größer oder viel kleiner als 31 ist, würde einen genetischen Unterschied zwischen den Sab Bondsmen und den Free-born Noble Somali andeuten. Benutze den Schieberegler unten, um den p-Wert zu erhalten.

Die Wahrscheinlichkeit so wenig wie 26 zu bekommen ist 0,1084. Da dies ein zweiseitiger Test ist, müssen wir auch die Wahrscheinlichkeit beachten eine Anzahl zu bekommen, die ungewöhnlich hoch ist, so dass der p-Wert doppelt ist, 0,2169. 26 Sab Bondsmen mit der Blutgruppe 0 zu bekommen ist daher nicht unwahrscheinlich, also schließen wir, dass es keinen Beweis aus diesen Daten für eine genetische Differenz zwischen Sab Bondsmen und den Free-born Somali gibt.
