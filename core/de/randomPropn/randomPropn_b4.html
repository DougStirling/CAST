Annahmen, die der Binomialverteilung zugrunde liegen

Die Binomialverteilung ist anwendbar auf eine große Anzahl von Anwendungen, die einer Zufallsstichprobe von kategorischen Messwerten entnommen sind. Die Trefferzahlen weisen eine Binomialverteilung auf, sofern ...

  • Jede Beobachtung dieselbe Wahrscheinlichkeit π für „Erfolg haben“ aufweist.
  • Jede der n Beobachtungen unabhängig entnommen wurde.
  • Wir die Trefferzahlen (oder -verhältnisse) für "Erfolg" der n Beobachtungen verzeichnet haben.

Auswertung von Binomialverteilungen

Falls wir überzeugt sind, dass eine Binomialverteilung angebracht ist, kann sie gewöhnlich unzählige Erfolgswahrscheinlichkeiten erzielen. Binomische Wahrscheinlichkeiten würden man erhalten durch...

  • Binomische Wahrscheinlichkeitstabellen,
  • eine mathematischen Formel, oder
  • einen Computer.

Zahlenbereich

Um die Wahrscheinlichkeit festzustellen, die die Trefferzahlen innerhalb irgendeines Intervalls repräsentieren, werden die Wahrscheinlichkeiten jeder Ganzen Zahlen innerhalb des Intervalls addiert.

Wenn das getan wird, muss der Wortlaut der Frage beachtet werden - denke sorgsam darüber nach, ob der„Extremwert“, der im Wortlaut des Intervalls genannt wird, mit einbezogen werden sollte.

In words...     In terms of X         Using 1/2    
More than 5 X > 5 X > 5.5
Greater than or equal to 5 X ≥ 5 X > 4.5
No more than 5 X ≤ 5 X < 5.5
At least 5 X ≥ 5 X > 4.5
Fewer than 5 X < 5 X < 4.5
5 or fewer X ≤ 5 X < 5.5

Die obere Tabelle gibt einige mögliche Intervallformulierungen in einem Zahlenbereich wieder. Die letzte Spalte liefert eine Auswertung jedes Intervalls, die sehr eindeutig darstellt, welche Zahlen enthalten sind.

Wir empfehlen jedes Intervall in seine Form zu übertragen, indem man ½ verwendet, bevor man die Wahrscheinlichkeit ermittelt.

(Diese Intervalltranslation ist teilweise sinnvoll, wenn die Normalennäherung verwendet wird, die auf den folgenden Seiten beschrieben wird.)

Beispiele

Jede Frage beinhaltet eine Binomialverteilung, dessen Balkendiagramm auf der linken Seite dargestellt ist. Und die Ganzen Zahlen, die näher in der Frage beschrieben werden, werden in verschiedenen Farben dargestellt. Die Summe dieser Wahrscheinlichkeiten (Balkenhöhen) ist mit dem Computer ausgewertet worden und wird auf der rechten Seite dargestellt, um die Antwort zu liefern.
