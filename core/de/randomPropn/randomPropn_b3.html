Anmerkungen

Wir verallgemeinern nun das Telepathieexperiment der vorangegangenen Seite. Erwäge eine unendliche kategorische Population, die ein Mengenverhältnis π einiger Kategorien enthält, welches wir „Erfolg“ nennen wollen. Wir nennen die anderen Werte in der Population 'Misserfolge'.

Im Beispiel der Telepathie könnte eine richtige Schätzung mit 'Erfolg' bezeichnet werden, eine falscher Rateversuch mit 'Misserfolg'. Die Wahrscheinlichkeit eines Erfolges ist  π = 0,333.

Die Bezeichnungen 'Erfolg' und 'Misserfolg' bieten Ausdrücke, die einen breiten Bereich von Datensätzen beschreiben können. Zum Beispiel,

Datensatz 'Erfolg'   'Misserfolg'  
  Geschlecht einer Stichprobe von Fischen   weiblich männlich
Qualität von exportierten Äpfeln gut beschädigt
Auswirkung eines Insektizids auf Käfer tot lebend

Wenn eine zufällige Stichprobe von n Werten aus dieser Population gewählt wird, bezeichnen wir die Anzahl der Erfolge mit x und das Verhältnis der Erfolge mit  x = np.

Verteilung eines Verhältnisses einer einfachen Zufallsstichprobe

Die Anzahl der Erfolge x, hat eine diskrete Standardverteilung, genannt Binomialverteilung, die 2 Parameter besitzt, n und  π. In praktischen Anwendungen ist n eine bekannte Konstante, aber π könnte unbekannt sein. Das Stichprobenverhältnis p, hat eine Verteilung der gleichen Form, wird aber durch n skaliert.

Mit geeigneter Auswahl der Parameter n und π kann die Binomialverteilung die Verteilung jedes beliebigen Verhältnisses einer Zufallsstichprobe beschreiben.

Form der Binomialverteilung

Das untenstehende Diagramm zeigt einige mögliche Formen der Binomialverteilung. Das Balkendiagramm hat doppelte Achsen und kann deshalb die Verteilungen von x und p zeigen.

Ziehe die Schieberegler, um die Parameter der Binomialverteilung anzupassen. Beobachten, dass

  • sich die Verteilung von p auf  π bezieht.
  • die Streuung der Verteilung von p mit der Zunahme von n abnimmt.
  • die Verteilung symmetrisch ist, wenn  π = 0,5; aber schiefer wird, wenn  π einen Wert zwischen 0 und 1 annimmt.

Das Diagramm kann benutzt werden, um binomische Wahrscheinlichkeiten zu erhalten. Dies geschieht durch das Festlegen geeigneter Werte für π und n und das anschließende Klicken auf einen der Balken des Balkendiagrammes.  

Telepathie-Experiment

Zum Beispiel: Um die Wahrscheinlichkeit einer Testperson zu finden, 4 von 5 Karten im Telepathie-Experiment richtig zu raten, setze π = 0.33 und n = 5, klicke dann auf den Balken für x =4. Die Wahrscheinlichkeit wird unter dem Balkendiagramm angezeigt.

Das nachstehende Diagramm demonstriert, dass eine Binomialverteilung in der Tat Variabilität zwischen Stichproben beschreibt. Das rosafarbene Balkendiagramm am unteren Ende des Diagrammes zeigt die Binomialverteilung mit den Parametern n = 20 und  π = 0.333. Dies beschreibt die Verteilung des Stichprobenverhältnisses der richtigen Rateversuche aus n = 20 Rateversuchen.

Klicke auf Akkumulieren und  nehme einige Stichproben. Beobachte, dass die Verteilung von x mit der theoretischen Binomialverteilung übereinstimmt. Wiederhole die Übung mit verschiedenen Stichprobengrößen.
