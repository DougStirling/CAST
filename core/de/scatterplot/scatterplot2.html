Scatterplots (Streudiagramme)

Ein Scatterplot stellt jedes Individuum als ein einzelnes Kreuz an einer vertikalen Achse (für die Variable Y) und an einer horizontalen Achse (für die andere Variable X) dar.

Per Definition wird die Variable an der vertikalen Achse Y und die Variable an der horizontalen Achse X genannt.

Beziehung zwischen Lebensspanne und Stoffwechselrate
