Signifikanzniveau

Die Entscheidungsregel beeinflusst die Wahrscheinlichkeit von Typ I- und Typ II- Fehlern, und es gibt immer einen Abwägungsprozess zwischen diesen beiden Wahrscheinlichkeiten. Einen kritischen Wert zu wählen, um eine Fehlerwahrscheinlichkeit zu reduzieren, erhöht automatisch die andere.

In der Praxis konzentrieren wir uns normalerweise auf die Wahrscheinlichkeit eines Typ I- Fehlers. Die Entscheidungsregel wird so gewählt, dass die Wahrscheinlichkeit eines Typ I- Fehlers gleich einem vorher gewählten Wert ist, oft 5% oder 1%. Diese Wahrscheinlichkeit wird das Signifikanzniveau des Tests genannt und die Wahl diese Niveaus sollte von der Art der Fragestellung abhängen. Je schlimmer die Konsequenzen, wenn H0 fälschlicherweise verworfen wird, desto geringer sollte das benutzte Signifikanzniveau gewählt werden.

Wenn das Signifikanzniveau eines Tests auf 5%  gesetzt wird, und wenn wir uns dann entscheiden, H0 zu verwerfen, dann sagen wir, dass H0 bei 5% Irrtumswahrscheinlichkeit verworfen wird.

Das Reduzieren des Signifikanzniveaus steigert die Wahrscheinlichkeit eines Typ II- Fehlers.

Die Wahl eines Signifikanzniveaus sollte von der Art der Fragestellung abhängen.

Je schlimmer die Konsequenzen, wenn H0 fälschlicherweise verworfen wird, desto geringer sollte das Signifikanzniveau gewählt werden. Bei vielen Anwendungen wird das Signifikanzniveau auf 5% festegelegt.

P-Werte und Entscheidungen

Den kritischen Wert für den Test einer Hypothese zu einem Populations-Mittelwert (mit bekannter Standardabweichung) mit beliebiger Irrtumswahrscheinlichkeit (z.B. 5% oder 1%) erhält man aus den Quantilen der Normalverteilungen. Für andere Tests von Hypothesen ist es möglich, ähnliche kritische Werte mit Quantilen der entsprechenden Verteilung der Teststatistik zu ermitteln.

Zum Beispiel, wenn man testet, ob das Mittel einer normalen Population, wenn die Standardabweichung der Population unbekannt ist, ist die Teststatistik ein t-Wert, und ihre kritischen Werte sind Quantile einer t-Verteilung.

Es scheint, dass eine andere Herangehensweise notwendig ist, um Entscheidungsregeln für unterschiedliche Tests von Hypothesen zu finden, aber das trifft nur zum Teil zu. Obwohl die zugrundeliegende Theorie vielfach vom Typ des Tests abhängt, kann die Entscheidungsregel für einen beliebigen Test auf seinem p-Wert basieren. Zum Beispiel ist die Entscheidungsregel für einen Test mit einer Irrtumswahrscheinlichkeit von 5% immer:

Entscheidung
p-Wert > 0,05     annhemen H0
p-Wert < 0,05     verwerfen H0

Für einen Test mit einer Irrtumswahrscheinlichkeit von 1%  sollte die Nullhypothese H0 verworfen werden, wenn der p-Wert unter 0,01 liegt.

Wenn ein Computerprogramm den p-Wert für einen Test einer Hypothese anbietet, kann er deshalb einfach übersetzt werden in Annahme (oder Verwerfen) der Null-Hypothese bei 5% oder 1% Irrtumswahrscheinlichkeit.
