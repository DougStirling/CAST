Interpretieren einer graphischen Zusammenfassung von einer
Stichprobe

Es gibt eine Variabilität zwischen den Stichproben in zusammengefassten Darstellungen von Stichproben einer Grundgesamtheit. In praktischen Situationen hat man jedoch nur einen einzelnen Datensatz (Stichprobe). Also wie kann man dieses Wissen für die Variabilität zwischen den Stichproben verwenden?

Man kann Merkmale wie Ausreißer, Cluster oder Schiefe in einem Datensatz abschätzen, indem man untersucht, wie oft sie in Zufallsstichproben von einer Grundgesamtheit ohne solche Merkmale auftauchen.  

Vergiftung von Seidenraupen

Eine früherer Abschnitt zeigte eine Tabelle, die die Überlebenszeit von 80 Seidenraupenlarven, die Natriumarsenat ausgesetzt wurden, darstellte.

Die erste Hälfte in diesem Diagramm zeigt einen Boxplot und einen verwackelten Dotplot der Überlebensdaten. Es gibt ein schwaches Anzeichen von Schiefe (ein langer Rest hin zu der Verteilung auf der rechten Seite). Weist das darauf hin, dass das Überleben eine geringe schiefe Verteilung hat oder kann es einfach ein Ergebnis der Stichproben von einer symmetrischen Population sein?

Man untersucht die Variabilität von ähnlichen Grafiken einer Normalverteilung mit demselben Mittelwert (272,6) und gleicher Standardabweichung (30,67) der Daten. Die untere Hälfte der Grafik zeigt eine solche zufällige Stichprobe von dieser Normalpopulation. Nun klickt man einige Male Stichprobe nehmen, um die Stichprobenvariabilität der Darstellung der Stichproben zu beobachten. 

Man beobachtet, dass so oft wie man den Eindruck einer Schiefe erhält, auch der Boxplot mit den eigentlichen Daten eine zeigt. Da der Grad der Schiefe so gebräuchlich ist, falls die Population eine symmetrische Normale ist, man kann folgern, dass dies kein Beleg dafür ist, dass die Überlebenszeit eine schiefe Verteilung hat.
