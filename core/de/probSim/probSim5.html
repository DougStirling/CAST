Zufallswerte

Das Ausführen einer Simulation von einem Wahrscheinlichkeitsmodell basiert auf der Erstellung von Zufallswerten von der Wahrscheinlichkeitsverteilung im Modell.

Ein Computerprogramm sollte normalerweise zur Entwicklung von Zufallswerten verwendet werden. Das Programm Excel beinhaltet Funktionen, die verwendet werden können.

Auf dieser Seite wird untersucht, wie man kategorische und numerische Werte von einer willkürlichen Verteilung erstellt, ohne auf eine Computer Software angewiesen zu sein.

Die Grundlage einer Zufallswerterstellung ist ein Zufallswert zwischen 0 und 1, für welchen jeder mögliche Wert gleich wahrscheinlich ist. Solch ein Wert soll von einer rechteckigen (oder gleichen) Verteilung zwischen 0 und 1 rühren und hat die Wahrscheinlichkeitsdichtefunktion, die man unterhalb sieht.

Ein Wert kann von einer rechteckigen Verteilung erstellt werden indem man stufenweise Zufallsziffern erstellt. (z.B. beim Werfen eines 10-seitigen Würfels).
