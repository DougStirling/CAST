Erstellen eines kategoriellen Werts

Das Erstellen eines zufälligen Werts von einer kategorischen Verteilung kann auf einem rechteckigen Verteilungszufallswert r bezogen werden. Falls die kategorische Verteilung zwei mögliche Werte hat, Erfolg und Misserfolg, und die Wahrscheinlichkeit von Erfolg als Symbol π bezeichnet wird, dann wird ein Erfolg erstellt werden falls r weniger als π ist.

Wenn es mehr als zwei mögliche Kategorien in einer Verteilung gibt, kann die Methode einfach erweitert werden. Jeder mögliche Wert entspricht einem Bereich der Werte von r, dessen Breite gleich der benötigten Wahrscheinlichkeit ist. (Man beachtet, dass alle Wahrscheinlichkeiten einer kategorischen Verteilung zu 1,0 addierst werden müssen).

Das folgende Diagramm zeigt eine Erstellung eines Zufallsergebnisses von einem Tennisspiel, in dem die Wahrscheinlichkeit, dass der Blaue Spieler den Weißen Spieler besiegt, 0,6 beträgt.

Klick Werte erstellen, um einen rechteckig verteilten Wert zu finden. Wenn dieser Wert weniger als 0,6 beträgt, ist ein kategorischer Wert Blau gewinnt erstellt; andererseits ist Weiß gewinnt erstellt.

Wiederholt man dies einige Male, stellt man fest, dass bei annähernd 60% der erstellten Werte Blau gewinnt.
