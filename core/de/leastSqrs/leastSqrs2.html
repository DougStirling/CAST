Einfache mathematische Form der Regressionskurve

Obwohl grundsätzlich jede Form von Regressionskurve zum Zeichnen per Augenmass möglich ist, ist es recht zweckmäßig sich auf einfache mathematische Funktionen zu beschränken

y  =  ƒ ( x )

Benutzt man ein Modell dieses Typs, kann man die abhängige Variable für jeden beliebigen Wert der erklärenden Variable durch Einsetzen des Wertes in die Funktion bestimmen. Das ist deutlich einfacher als den Wert, von der per Augenmaß gezeichneten Kurve abzulesen.

Lineare Gleichungen

Obwohl einige Zusammenhänge durch Kurven beschrieben werden müssen, liefert eine Gerade in vielen Fällen eine adäquate Beschreibung bivariater Datensätze. Eine Gerade hat folgende Gleichung

y  =  b0 + b1 x

Die Konstante b0 bezeichnet man als Achsenabschnitt, sie beschreibt den y-Wert wenn der x-Wert gleich Null ist. Die Konstante b1 wird als Steigung bezeichnet; Sie beschreibt die Veränderung von y wenn x um eine Einheit ansteigt.

Benutzt man eine Gerade zur Beschreibung eines Zusammenhangs zwischen zwei Variablen, so sprechen wir von einem linearen Modell oder einem linearen Regressionsmodell.

Vorhersagen können mit einem linearen Modell sehr einfach gemacht werden.

Ein lineares Modell kann benutzt werden um den Wert von y für jeden beliebigen Wert von x vorherzusagen. Dazu benutzt man die Gleichung

  =  b0 + b1 x
