Kategoriale Daten verschiedener Gruppen

Liegt eine einzelne kategoriale Variable vor, so kann man mit einem Bakendiagramm oder einem Kreisdiagramm sehr leicht Informationen deutlich machen. Interessanter wird es meist jedoch, wenn die Daten von verschiedenen Gruppen stammen.

  • Unterscheiden sich die Krebsarten bei bestimmten Berufsgruppen?
  • Bekommen ältere Studierende bessere Noten (A, B, C or Fail), als jüngere Studierende an der Universität?
  • Wie effektiv schützen verschiedene Pflanenschutzbehandlungen vor Schadbefall?

Bei allen Fragestellungen gibt es eine kategoriale Variable (Krebsart, Note, Schadbefall, ...) und verschiedene Gruppen (Berufsgruppe, Altersgruppe, Pflanzenschutzbehandlung, ...).

Kontingenztabelle

Die kategorialen Variablen können, einzeln für jede Gruppe, in einer Häufigkeitstabelle dargestellt werden. Ordnet man diese Häufigkeitstabellen nebeneinander in einer Tabelle an, so erhält man eine Kontingenztabelle (Kreuztabelle).

Umfrage unter Reisbauern

Im Rahmen einer Umfrage wurden 36 Bauern aus 4 Dörfern befragt. Jeder Bauer gab an, welche Reissorte er anbaute. Die Sorten wurden in die Kategorien "Alt", "Neu" oder "Traditionell" eingeordnet. Die 36 Werte wurden nach Dörfern gruppiert (siehe Diagramm unten links).

Klicke auf die Werte von Sabey, um sie in der ersten Spalte der Kontingenztabelle rechts einzufügen. Füge auch die Werte der anderen Dörfer in die Kontingenztabelle ein.

Möglicherweise liegen die Daten nicht in Form von seperaten Listen für jedes einzelne Dorf vor, sonder sind in einer Datenmatrix eingegeben. In diesem Fall stellt die Gruppenvariable "Dorf" eine zusätzliche kategorielle Variable dar. Jede Beobachtungseinheit (Bauer).
