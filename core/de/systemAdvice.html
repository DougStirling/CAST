Ein Update auf CAST ist verfügbar

Nachdem sichergestellt ist, dass Java installiert ist, führen die CAST_manager.jar -Programm in der CAST-Ordner zu aktualisieren CAST auf die neueste Version.

Einstellungen für unter (lokale)

Java

Die interaktive Diagramme in CAST sind Java-Programme. Da Java nicht auf Android oder Apple-Tabletten verwendet werden, sind die interaktiven Versionen von CAST Seiten daher nicht zur Verfügung.

Obwohl Sie in der Lage sein, um die Seiten in diesem E-Book finden, ihre interaktiven Diagramme leer sein. Seit CAST so aufgebaut, dass statistische Konzepte mit seinen interaktiven Diagramme erklären, werden Sie nicht finden es einfach zu lesen.

Da diese E-Buch hat Video-oder Zusammenfassung Versionen seiner Seiten, werden Sie in der Lage sein, sie zu lesen, aber die interaktive Versionen der Seiten werden nicht zur Verfügung.

Da die interaktive Versionen der CAST Seiten ein so wichtiger Teil des E-Book sind, empfehlen wir, dass es auf einem vollständigen Computer (Laptop oder Desktop) eher als ein Tablet oder Smartphone wird untersucht.

Die interaktive Material in CAST basiert auf Java, so ist es wichtig, zu installieren und zu aktivieren.

Wenn ein grünes Rechteck erscheint ohne Warnungen erwähnt, wird Java funktioniert prima. (Es kann ein oder zwei Minuten zu erscheinen.)
Wenn sie nicht erscheint, zunächst sicherstellen, dass Sie die neueste Version von Java installiert haben.

Auch nach Java installiert wurde, müssen Sie die Einstellungen in Ihrem Betriebssystem oder Browser zu ändern, um es vollständig zu aktivieren (und sehen das grüne Rechteck oben ohne Warnungen).

Aktivieren Java 7

Wenn Sie die neueste Version von Java (Java 7), muss es in der Windows Systemsteuerungg aktiviert werden. In der Windows-8-Desktop können Sie die Systemsteuerung, indem Sie entweder mit der rechten Maustaste den unteren linken Ecke des Bildschirms zugreifen oder halten Sie die Windows-Taste und die Taste 'X'. In Windows 7 oder früher, wird die Systemsteuerung über das Startmenü aufgerufen.

Wenn das Control Panel sichtbar ist, wählen Sie die Java Control Panel , dann klicken Sie auf die Sicherheit . Im Erweitert , reduzieren Sie die Sicherheitsstufe von Hoch bis Low Medium .

Internet Explorer Einstellungen

Internet Explorer 8 und früher nicht unterstützt Web-Standards für Video, so dass die Videos, die in diesem E-Book sind nicht beobachtet werden. Um die Videos zu sehen, empfehlen wir entweder mit Firefox oder Chrome ein Upgrade auf Internet Explorer 9. Die ursprüngliche interaktives Material in CAST noch funktioniert.

Klicken Sie auf die Werkzeuge -Schaltfläche und dann auf Internet Optionen .

  • Im Erweitert Registerkarte wählen Sie die Option Ausführung aktiver Inhalte in Dateien auf diesem Computer .
  • Im Sicherheit , schalten off die Option Geschützten Modus aktivieren .

Firefox-Einstellungen

Bei Verwendung eines lokal installierte Kopie von CAST, Firefox zeigt immer eine Warnung statt jedem interaktiven Diagramm (Java-Applet). Als Ergebnis muss jedes Diagramm angeklickt werden, um diese Warnung zu entlassen, bevor sie angezeigt werden.

Es gibt derzeit keine Möglichkeit, Firefox zu sagen, Java global aktivieren, wenn Sie eine lokale Kopie der CAST, so empfehlen wir die Verwendung eines anderen Browsers.

Standardmäßig Firefox zeigt immer eine Warnung statt jedem interaktiven Diagramm (Java-Applet) in CAST. Einzelne Diagramme können angeklickt werden, um diese Warnungen entlassen werden.

Diese Warnungen kann global für CAST abzuweisen. Wenn eine Seite mit einem Applet angezeigt wird, wird ein rotes 'Lego block' neben der URL oben links im Fenster angezeigt. Klicken Sie hier, um eine Nachricht zu Fenster über die Java-Plugin angezeigt. Klicken Sie auf das aktivieren Alle Plugins im Dropdown-Menü und wählen Sie Immer aktivieren Plugins für diese Seite .

Chrome-Einstellungen

Wenn Chrome zuerst einen CAST Seite zeigt mit einer interaktiven Grafik, fragt er am oberen Rand des Fensters für die Erlaubnis, Java laufen.

Wenn Sie auf immer auf dieser Seite ausführen , Chrome zeigt die interaktive Diagramme in CAST für den Rest dieser Sitzung. Da es sich um eine lokal installierte Kopie von CAST wird jedoch Chrome wieder fragen, wann CAST später neu gestartet wird.

Chrome legt zusätzliche "Sicherheit" Einschränkungen bei der Ausführung eine lokale Kopie von CAST und diese verhindern, dass der Index und die Liste der Datensätze aus der Kommunikation mit den Hauptdarstellern Fenster. Wenn Sie diese Funktionen in CAST verwenden möchten, empfehlen wir den Internet Explorer oder Firefox.

Wenn Sie auf immer auf dieser Seite ausführen , Chrome zeigt die interaktive Diagramme in CAST von da an.

Safari-Einstellungen

Beim Ausführen von CAST von einem lokal installierten Kopie muss ein paar Änderungen an der Safari Einstellungen vorgenommen werden, bevor es CAST interaktiven Diagrammen (Java-Applets) wird angezeigt.

  • Wählen Sie Einstellungen aus dem Safari-Datei Menü.
  • Im Erweitert Einstellungen, wählen anzeigen Entwickeln in der Menüleiste . Safari wird nun eine extra Menü namens Entwicklung in der Menüleiste.
  • In der Menü Entwickeln, deaktivieren Sie die Option Local File Einschränkungen ist für die interaktive Diagramme eingestellt werden, erscheinen.

Safari auf Lion oder eine neuere Version von MacOS X auferlegt zusätzliche "Sicherheit" Einschränkungen bei der Ausführung eine lokale Kopie von CAST und diese verhindern, dass der Index und die Liste der Datensätze aus der Kommunikation mit den Hauptdarstellern Fenster. Wenn Sie diese Funktionen in CAST verwenden wollen, empfehlen wir die Verwendung Firefox.

Standardmäßig zeigt Safari eine Pop-up-Warnung, wenn Sie zu jeder Seite mit CAST ein interaktives Diagramm aussehen, gefragt, ob Sie zulassen möchten oder blockieren es.

  • Klicken Sie auf Allow.
  • Wählen Sie Einstellungen aus dem Safari-Datei Menü.
  • Im Sicherheit Einstellungen, um sicherzustellen, dass Erlauben Java markiert ist, klicken Sie dann auf verwalten Website-Einstellungen.
  • In dem Dialogfeld, das angezeigt wird, gibt es einen leeren Eintrag, die lokal installierten Dateien darstellt. Ändern Sie die Einstellungen auf Allow Always.
  • In dem Dialogfeld, das angezeigt wird, gibt es einen Eintrag entsprechend dem CAST Website. Ändern Sie die Einstellungen auf Immer zulassen.
