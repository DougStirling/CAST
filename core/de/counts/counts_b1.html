Diskrete und kontinuierliche Daten

In diesem Abschnitt unterscheiden wir zwischen zwei Arten von numerischen Daten.

Diskrete Daten
Wenn die Werte einer Menge ganze Zahlen sind (Zähldaten), nennt man den Datensatz diskret. Beispiele für diskrete Messungen sind: 
  • Die Zahl der Raupen auf jedem Kohlkopf in einem Feld,
  • die Zahl der gezogenen Zähne von Patienten im Alter von 50-60 Jahren einer Zahnarztpraxis,
  • die Zahl der gequetschten Äpfel in jeder der 20 Kisten mit je 100 Äpfeln,
  • die Zahl der Eier gelegt von einer bestimmten Vogelart,
  • die Zahl der täglichen Einweisungen einer Notaufnahme eines Krankenhauses über einen Zeitraum von zwei Monaten.
Kontinuierliche Daten
Wenn die Daten nicht auf ganze Zahlen beschränkt sind, nennt man den Datensatz kontinuierlich.Beispiele sind:
  • Die Gewichte der von einem Fischerboot gefangenen Kabeljaue,
  • der Ertrag von Weizen (in Tonnen pro Hektar) in einer Stichprobe von Betrieben aus einer Region,
  • die Konzentration eines Schadstoffes in jeder der 20 Proben von Flusswasser, die über einen Zeitraum von einer Woche genommen wurden,
  • den Cholesterinspiegel von 50 Patienten, die mit leichtem Herzinfarkt ins Krankenhaus eingeliefert wurden.
     

Dotplots for Zähldaten

Punktdiagramme können verwendet werden, um Zahlendaten anzuzeigen. Wenn allerdings die Zähldaten sehr klein sind, erscheinen viele der Werte in der Regel mehrere Male. Einfache Punktdiagramme sind daher irreführend, da wiederholte Werte überlagert werden und als ein einziges Kreuz erscheinen.

Allerdings können gestapelte und verwackelte Punktdiagramme trotzdem verwendet werden. Wenn die Zähldaten alle klein sind, braucht keine Information durch Überlagerung verloren zu gehen, da dort ein Balken von Kreuzen für jeden Wert entstehen kann.
