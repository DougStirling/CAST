Beschreibung der Glaubhaftigkeit von Null-Hypothesen

Auf der vorhergehenden Seite beschreibt ein Diagramm mit Skalen, wie der Beweis gegen H0, für verschiedene Datensätze „geprüft“ wird.
Ein p-Wert ist eine numerische Beschreibung von diesem Beweis, der ein Maßstab zu diesem Diagramm geben kann.

Ein P-Wert ist eine numerische zusammenfassende Statistik, die den Beweis gegen H0 beschreibt.

Effekt von Pheromonen auf Kanninchen

Ein Experiment wird durchgeführt, um abzuschätzen, ob Kanninchen von bestimmten Pheromonen beeinflusst werden, entweder positiv oder negativ. Ein großer Käfig wird konstruiert, dessen Funktionen (Wasser, Nahrung etc.) auf beiden Seiten einer Zentrallinie symmetrisch dupliziert werden. Ein Kanninchen wird in der Mitte des Käfigs plaziert und eine kleine Menge von Pheromonen wird auf einer Seite ausgebracht. In den nächsten fünf Stunden nimmt ein Techniker die Anzahl von Minuten auf, die das Kanninchen auf jeder Seite des Käfigs verbringt. Das Experiment wird mit 16 Kanninchen wiederholt.

Tendieren die Kanninchen dazu die Hälfte der Zeit auf jeder Seite des Käfigs zu verbringen (was dafür spräche, dass sie von den Pheromonen unbeeinfusst sind) oder tendieren sie dazu, mehr Zeit auf der einen oder anderen Seite zu verbringen (was dafür spräche, dass Anziehung oder Abstoßung vorliegt)? Wir begutachten diese Frage, indem wir den Unterschied zwischen den Aufenhaltszeiten auf beiden Seiten benutzen, Pheromon minus kein Pheromon. Diese Frage entspricht der Frage, ob ein Beweis vorliegt, dass der zugrunde liegende Mittelwert der Population des Unterschiedes Null sein könnte.

Das unten stehende Diagramm prüft den Beweis, unter Benutzung des p-Wertes eines t-Tests mit µ = 0.


Der p-Wert ist Index für die Glaubwürdigkeit der Null-Hypothese, µ = 0.

P-Werte haben die gleiche Interpretation für alle Hypothesen-Tests.
