Unendliche Grundgesamtheiten

Bei der Vorstellung, dass einem Datensatz eine abstrakte Grundgesamtheit zugrunde liegt, enthält diese in der Regel eine unendliche Anzahl von Werten. Als Beispiel kann die Lebensdauer einer Stichprobe von Glühbirnen betrachtet werden. Die Grundgesamtheit von möglichen Ausfallzeiten enthält alle Werte größer als Null, und beinhaltet auch eine unendliche Anzahl von Werten.

Der Grundgedanke des Stichprobenziehens aus einer unendlichen Grundgesamtheit ist schwierig, so dass er jetzt in einem anderen Zusammenhang als eine Erweiterung des Stichprobenziehens aus einer endlichen Grundgesamtheit dargestellt wird.

Position von Kühen innerhalb eines Feldes

Man kann sechs Kühe betrachten, die sich frei innerhalb eines Feldes bewegen können. Ihre Position in dem Feld wird zu einem bestimmten Zeitpunkt betrachtet, so dass die Daten sechs „Positionen“ der Kühe darstellen.

Zunächst muss berücksichtigt werden, dass das Feld in ein 5x5-Gitter eingeteilt ist, was eine Grundgesamtheit von 25 möglichen Positionen für die Kühe ergibt.  Die Position der Kühe ist ein zufälliges Ziehen einer Stichprobe von sechs von der Grundgesamtheit. Beim mehrmaligen Klicken auf Stichprobe ziehen, können mit diesem Modell mögliche Positionen betrachtet werden.

Man kann das Pop-Up-Menü verwenden, um das Gitter in ein 10x10-Gitter zu ändern und dann in ein 30x30-Gitter, um eine genauere Angabe der Kuh-Positionen zu ermöglichen. In beiden Fällen werden immer noch Stichproben aus einer endlichen Grundgesamtheit erhoben. 

Schließlich kann man Unendlich aus dem Pop-Up-Menü auswählen, um das Raster bin ins Äußerste zu verfeinern, so dass die Positionen der Kühe überall in diesem Feld sein können — eine unendliche Grundgesamtheit. Bei Klicken von Stichprobe auswählen wird eine zufällige Stichprobe von Positionen dieser unendlichen Grundgesamtheit ausgewählt.
