Breite eines 95% Konfidenzintervall für den Mittelwert

Auf der vorherigen Seite wurde demonstriert, dass eine Intervallschätzung berechnet mit

eine geringeres Konfidenzniveau als 95% hat. Um ein Konfidenzniveau von 95% zu erreichen, muss das Intervall erweitert werden.
Das kann durch das Ersetzten von 1,96 durch eine etwas größere Zahl erreicht werden. Wir werden versuchen

wobei k > 1,96.

Simulation

Das unten stehende Diagramm zeigt zunächst eine zufällige Stichprobe von n = 30 Werten einer normalen Grundgesamtheit (Population) mit µ = 10 und σ = 2.

Wenn k = 1,69 ist, ist das Konfidenzniveau 94%. Man benutzt den Schieberegler, um den Wert k so anzupassen, dass das Konfidenzniveau 95% ist. (Die Pfeiltasten auf der Tastatur können für die Feineinstellung von k in Schritten von 0,01 benutzt werden.)

Das resultierende Konfidenzintervall von 95% wird in rot hinter den Daten dargestellt. Man klickt weitere Stichprobe, um 95% Konfidenzintervalle für andere Datensätze zu erhalten. 

Benutze das Pop-Up-Menü, um die Stichprobengröße auf 5 zu verändern und wiederhole das Ganze. Man beobachte, dass k erhöht werden muss, um ein Konfidenzniveau von 95% zu erreichen. 

Der Wert von k, der für ein 95% Konfidenzniveau sorgt, hängt von der Stichprobengröße ab.

Der t-Wert 

Der Wert k, der benötigt wird, um ein 95% Konfidenzniveau zu erhalten, hängt von der Stichprobengröße n ab und wird gekennzeichnet durch tn-1. Aus Gründen, die hier nicht erklärt werden können, muss der t-Wert bei einer Stichprobengröße minus eins abgelesen werden, statt bei der tatsächlichen Stichprobengröße. Der Wert n - 1 wird Freiheitsgrad der Konstanten genannt. 

Finden des t-Wertes 

Man benutz oft eine gedruckte Tabelle oder Computersoftware, um den t-Wert für ein Problem zu erhalten. Das untenstehende Diagramm zeigt, wie der t-Wert von dem Freiheitsgrad (und daher von der Stichprobengröße) abhängt.

Man zieht den Schieberegler, um den Freiheitsgrad zu verändern und liest die entsprechende t-Werte ab. Merke, dass

  • Der t-Wert, der einem Freiheitsgrad von 1 entspricht (und daher einer Stichprobengröße von 2 hat) zu groß ist, um auf der Grafik abgebildet zu werden.
  • Wenn die Freiheitsgrad größer als 100 sind, ist der t-Wert unter 2,0. Wenn man den Freiheitsgrad auf 100000 verändert, ist der t-Wert 1,960.
  • Der Wert 2,0 ist eine angemessene Annäherung an den t-Wert, wenn der Freiheitsgrad größer als 30 ist.
