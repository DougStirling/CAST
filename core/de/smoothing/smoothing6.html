Örtliche Glättung von Streudiagrammen

Lowess kann dazu benutzt werden, Zeitreihen zu glätten, war aber eigentlich als allgemeiner Weg, um eine Glättungskurve für jeden Typ von Streudiagramm zu zeichnen, entwickelt worden. Erneut wird der geglättete Wert für jede Beobachtung durch Anpassung einer kleinsten Quadrate Linie an die Beobachtungen mit benachbarten Werten für die erklärende Variable erhalten. Die angepassten Werte, die auf diese Weise erhalten worden sind, sind mit Linien verbunden.

( Die meiste Computer Softwares implementiert eine Version von Lowess, die tatsächlich etwas komplexer ist, als sie hier beschrieben worden ist, aber unsere einfachere Version gibt einen guten Eindruck von der Methode.)

Mehltau bei Rosen

Das folgende Streudiagramm zeigt die Ergebnisse eines Experiments, das durchgeführt wurde, um zu bestimmen, wie die Latenzzeit ( Zeit in Tagen von der Erstinfektion bis zur Sporulation) von Mehltau bei Rosen abhängig von der Temperatur ist.

Ziehe, wie auf der vorherigen Seite, über die Kreuze, um die benachbarten Werte, die für die kleinsten Quadrate Linie, die jeden geglätteten Wert liefert, benutzt werden, zu sehen. 

Benutze den Schieberegler, um die Fensterbreite anzupassen. Fenster mit etwa zehn benachbarten Punkten ergeben eine ziemlich glatte Kurve.
