Beschreiben der Genauigkeit mit einer Intervallschätzung

Die Verwendung eines einzelnen Wertes als Parameterschätzung (eine Punktschätzung) liefert keine Informationen über die Genauigkeit des Schätzers – d.h. über die voraussichtliche Größe des Schätzfehlers.

Eine bessere Möglichkeit, ist es eine Parameterschätzung mit einer Intervallschätzung darzustellen. Eine Intervallschätzung zeigt den Grad der Unsicherheit des Wissens über den Wert des Parameters. Man bevorzugt zum Beispiel eine Aussage wie:

 µ liegt wahrscheinlich zwischen 10,5 und 11,5.

eher als einfach

unsere Schätzung von µ ist 11,0.

Werden die Intervallschätzungen den tatsächlichen Wert des Parameters beinhalten?

Man betrachtet eine Intervallschätzung des Mittelwerts der Grundgesamtheit, μ, zentriert auf den Mittelwert einer Zufallsstichprobe, x. Da man die Breite dieser Intervallschätzung reduziert, bekommt man weniger den Eindruck, dass das Intervall den unbekannten Wert von μ enthält.

Später wird beschrieben wie man das Vertrauen messen kann, dass eine bestimmte Intervallschätzung einen Parameter wie μ enthalten kann.
