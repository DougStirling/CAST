Standardabweichungen vom Mittelwert

Alle Normalverteilungen haben die gleiche Form an einer Skala von "Standardabweichungen vom Mittelwert".

Diese Gleichung kann auch in folgender Form geschrieben werden

x  =  μ  +  z × σ

Wahrscheinlichkeiten und Z-Werte

Die Eigenschaften von Normalverteilungen auf der vorherigen Seite geben die Wahrscheinlichkeiten, dass ein Z-Wert innerhalb ±1, ±2 und ±3 liegt:

  • P(-1 < z < +1) ist etwa 0,68
  • P(-2 < z < +2)  ist etwa 0,95
  • P(-3 < z < +3)  ist etwa 0,997

Jede andere Wahrscheinlichkeit (Fläche) in Bezug auf eine normalverteilte zufällige Variable X kann in Termen von z-Werten gefunden werden:

  1. Übersetze die x-Werte, deren Wahrscheinlichkeiten gebraucht werden, in einen z-Wert, dann
  2. Nutze den z-Wert, um die relevante Fläche in dem obigen Diagramm zu bestimmen.

(Wir werden auf der folgenden Seite erklären, wie Wahrscheinlichkeiten von z-Werten genau bestimmt werden können.)

Gewichte von Äpfeln

Eine Fabrik, die Äpfel verpackt, hat beobachtet, dass Fuji-Äpfel von den Zulieferern etwa einer Normalverteilung mit einem Mittelwert von 180g und einer Standardabweichung von 10g gleichen.

Das obige Diagramm überträgt das ausgewählte Apfelgewicht x in einen z-Wert, welcher beschreibt, wie viele Standardabweichungen (in Einheiten von 10g) über dem Mittelwert (180g) liegen.

Der z-Wert definiert die Position des Apfelgewichts auf der unteren der beiden Achsen. Die hervorgehobene Fläche ist die Wahrscheinlichkeit eines geringeren Wertes.
