Finden eines x-Wertes zu einer Wahrscheinlichkeit

Die vorherigen Seiten haben erklärt, wie eine Wahrscheinlichkeit, dass ein Wert einer Normalverteilung kleiner als ein Wert x ist, gefunden werden kann.

Unter manchen Umständen müssen wir ein inverses Problem lösen – wir haben eine Wahrscheinlichkeit gegeben und müssen den Wert x finden, so dass es eine Wahrscheinlichkeit gibt, geringer zu sein.

Begriffsdefinitionen

Quartile
Die Quartile einer Verteilung sind die drei Werte, so dass eine Wahrscheinlichkeit  von ¼, ²⁄â‚„ und ¾  besteht, niedriger zu sein.
Perzentile
Das r`te Perzentil einer Verteilung ist der Wert mit der Wahrscheinlichkeit r/100 niedriger zu sein.
Quantile
Diese sind verallgemeinert in dem Term Quantil. Der Wert mit der Wahrscheinlichkeit p niedriger zu sein, nennt man Quantil der Verteilung entsprechend der Wahrscheinlichkeit p.

Gewichte von Äpfeln

Das Diagramm unten zeigt die Verteilung des Gewichtes von Fuji-Äpfeln, die am Packhaus ankommen. Die Verteilung ist normal (µ = 180g, σ = 10g).

Der Schieberegler übersetzt das Apfelgewicht x in z-Werte und nutzt die z-Tabellen, um die Wahrscheinlichkeit für einen Apfel mit geringerem Gewicht als x zu finden.

Die größten 10% der Äpfel werden zum Export verkauft. Wie groß sind diese Äpfel?

Die Frage sucht das Gewicht x, so dass

P ( Apfelgewicht < x )   =   0,9

Passe den Schieberegler auf die Wahrscheinlichkeit 0,9 an.

Quantile finden

Die oben aufgeführte 'trial-and-error' Methode ein Quantil zu finden, beinhaltet das Probieren von verschiedenen x-Werten bis die Zielwahrscheinlichkeit erlangt ist.

x probability

Eine bessere Methode benutzt den umgekehrten Ablauf,

probability x

Der erste Schritt dieses Prozesses beinhaltet das Finden der Z-Tabelle, für welche die benötigte Wahrscheinlichkeit niedriger ist. Statistische Software oder Excel kann diesen z-Wert berechnen, oder man kann statistische Tabellen verwenden. Zum Beispiel zeigt das Diagramm unten, wie man die Z-Tabelle findet, so dass eine Wahrscheinlichkeit von 0,9 dafür gibt niedriger zu sein.

Übersetzten von einer Z-Tabelle in die dazugehörigen x-Werte kann man mithilfe einer Formel gemacht werden,

x  =  μ  +  z σ

(Erinnere dich, dass die Z-Tabelle angibt, wie viele Standardabweichungen man vom Mittelwert entfernt ist.)
