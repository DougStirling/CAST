Kommen die Daten aus einer Normalverteilung?

Ein Histogramm der Daten kann überprüft werden und kann andeuten, dass es eine Schiefe gibt oder, dass sich die Verteilung in Cluster aufteilt. Wenn der Datensatz jedoch groß ist, kann eine Normalwahrscheinlichkeitsfläche mehr feine Abweichungen von einer Normalverteilung andeuten.

Normalwahrscheinlichkeitsfläche

Eine Normalwahrscheinlichkeitsfläche ist nach dem folgenden Weg angefertigt:

  1. Sortiere die Datenwerte in eine Reihenfolge, x(1) < x(2) < ... < x(n)
  2. Finde geordnete Werte, die so breit sind, wie du es von einer Normalverteilung erwartest, q1 < q2 < ... < qn. Die Quantile der Normalverteilung beziehen sich auf Wahrscheinlichkeiten , 1/(n+1), 2/(n+1), ..., n/(n+1), …, die im Allgemeinen verwendet werden.  
  3. Plotte x(i) gegen qi

Wenn der Datensatz von einer Normalverteilung ist, sollten die Daten in einem ähnlichen Weg zu den Normalquantilen verteilt werden, so dass die Kreuze in der Normalwahrscheinlichkeitsfläche nahe einer Geraden liegen.

Beispiele

Interpretieren der Form einer Normalwahrscheinlichkeitsfläche ist leichter mit einigen Beispielen zu erklären.

Die Werte auf der horizontalen Achse sind q1 < q2 < ... < qn, welche soweit verteilt sind, wie du es von einer Normalverteilung erwarten würdest. Solche auf der vertikalen Achse sind die aktuellen Datenwerte. Datensätze mit verschiedenen Eigenschaften können vom Pop-Up-Menü ausgewählt werden.

Beobachte, wie die Verteilung des Datensatzes die Form der Wahrscheinlichkeitsfläche beeinflust.

Wie viel Krümmung wird benötigt, um eine Nicht-Normalität anzudeuten?

In den obigen Beispielen war Linearität und Nichtlinearität in der Wahrscheinlichkeitsfläche klar. In der Praxis jedoch bedeutet die Zufälligkeit der Daten, dass die Wahrscheinlichkeitsfläche nicht genau gerade sein wird, auch wenn die Daten aus einer Normalpopulation stammen.
