Schätzung des Mittelwerts einer Population

Ein Mittelwert einer Population, μ, wird in der Regel durch den Mittelwert aus einer Zufallsstichprobe geschätzt.

Wir untersuchen nun die Fehlerverteilung für diesen Schätzer.

Verteilung des Schätzers

Die Verteilung der Schätzfehler

Der Fehler in der Schätzung ist:

Fehler   =    − µ

so dass er zwar die gleiche Verteilung wie x hat, aber so verschoben ist, dass er auf Null zentriert ist. Daher ist die Fehlerverteilung annähernd:.
