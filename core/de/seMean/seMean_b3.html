Interpretation des Standardfehlers

Die 70-95-100 Faustregel bietet eine nützliche Interpretationshilfe des Standardfehlers eines Schätzers. Für alle Variablen mit einigermaßen glockenförmigen Verteilungen,

beträgt die Wahrscheinlichkeit ca. 95%, dass innerhalb von 2 Standardabweichungen des Mittelwertes ein Wert erreicht wird und es ist nahezu sicher, dass ein Wert innerhalb von 3 Standardabweichungen des Mittelwertes erreicht ist. 

Wird dies auf eine Fehlerverteilung eines Schätzers angewandt und angenommen, dass die Verzerrung Null (oder zumindest klein) ist, 

so beträgt die Wahrscheinlichkeit ca. 95%, dass der Fehler innerhalb 2 Standardabweichungen von Null liegt und höchstwahrscheinlich innerhalb von 3 Standardabweichungen von Null liegt.

Beachte, dass der Begriff Standardfehler oft mit SE abgekürzt wird.
