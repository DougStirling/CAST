Eine Zufallsstichprobe ohne komplizierte Hilfsmittel ziehen

Zieht man eine Zufallsstichprobe, so muss sichergestellt werden, dass jedes Element der Grundgesamtheit die gleiche Chance hat in die Stichprobe zu gelangen. Wie kann solch eine zufällige Stichprobe in der Praxis durchgeführt werden? Eine Methode eine Stichprobe mit dem Umfang n zu ziehen ist...

  1. schreibe den Namen (oder eine andere Identifikationseigenschaft) aller Elemente der Grundgesamtheit auf gleiche Papierstücke,
  2. mische die Papierstücke (Lose) in einer Schachtel
  3. ziehe n Papierstücke (mit oder ohne Zurücklegen).

Für große Grundgesamtheiten ist das nicht durchführbar.

Zufallszahlen

Eine alternative Methode ist die Verwendung von Zufallszahlen. Es gibt eine Vielzahl an Möglichkeiten um Zufallszahlen zu generieren, wobei die Chancen für die Zahlen jeweils gleich groß sein müssen.

  • Würfeln mit einem 10-seitigen Würfel. (6-seitige Würfel sind normal, 10-seitige erlauben aber eine breitere Anzahl an Zufallszahlen.)
  • Es gibt Bücher mit Zufallszahlen. Öffne das Buch zufällig auf einer Seite und tippe mit geschlossenen Augen auf einen Wert. 
  • Einige Computerprogramme (z.B. Microsoft Excel und R) beinhalten Funktionen, die Zufallszahlen erzeugen können.
  • Viele Taschenrechner haben eine Zufallszahl-Taste.

Das Zusammenführen von 2 oder mehr Zufallszahlen ergibt größere Zufallszahlen (besonders interessant beim 10-seitigen Würfel).

Klicke Generate digit um eine Zufallsziffer zu erzeugen.

Fasst man 3 solcher Ziffern zusammen ergibt sich eine Zufallszahl zwischen 0 und 999. Klicke Generate value um eine Zufallszahl in diesem Variationsbereich zu finden.

Stichprobe wählen

Um eine Zufallsstichprobe mithilfe von Zufallszahlen zu ziehen sind folgende Schritte nötig:

  1. Nummeriere alle Elemente der Grundgesamtheit aufsteigend von 0.
  2. Erzeuge eine Zufallszahl mit so vielen Ziffern, wie die größte vergebene Nummer aufweist.
  3. Weist die Zufallszahl auf ein Element, dass noch nicht in der Stichprobe ist, dann füge es zu der Stichprobe hinzu. (Ansonsten ignoriere die Zufallszahl und erzeuge eine Neue.)
  4. Wiederhole Schritt 2. und 3. bis genügend Elemente in der Stichprobe sind.

In Schritt 3 können Zufallszahlen aus zwei Gründen abgelehnt werden.

  1. Die Zahl gehört zu keinem Element der Grundgesamtheit (z.B. die Zufallszahl ist 653, aber die Elemente haben Nummern von 0 bis 357)
  2. Das Element wurde bereits gewählt.

Der Punkt 1 kommt nur vor, wenn die Zufallszahlen über den tatsächlich vergebenen Bereich hinausreichen. Die Häufigkeit von Punkt 2 hängt davon ab, wie groß die Stichprobe im Vergleich zur Grundgesamtheit ist.

Das folgende Diagramm illustriert die Stichprobenziehung aus einer Grundgesamtheit von 56 Individuen, die von 0 bis 55 durchnummeriert wurden.

Klicke Random index. Ist der Zufallswert zwischen 0 und 55, wird das entsprechende Individuum zur Stichprobe hinzugefügt. Falls der Wert über 55 liegt, muss eine neue Zufallszahl generiert werden.

Wiederhole der Vorgang mehrmals. Du wirst feststellen, dass

  • beinahe die Hälfte der Zufallszahlen ungültig sind, weil kein Individuum den entsprechenden Index aufweist
  • ab und zu die Zufallszahlen auf Individuen verweisen, die bereits in der Stichprobe sind. Je größer die Stichprobengröße, desto häufiger kommt das vor.
