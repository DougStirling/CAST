Modell ohne Interaktion zwischen den Faktoren

Das einfachste Modell für den Effekt zweier Faktoren auf eine abhängige Variable ist ein Zusätzliches mit folgender Form:

(Durchschnittsergebnis)   =   (Basiswert)   +   (Effekt von Faktor A)   +   (Effekt von Faktor B)

Eine Folge dieses Modells ist, dass bei Veränderung von Faktor A der Effekt auf das Ergebnis der gleiche bleibt, unabhängig von der Höhe des Faktors B. Umgekehrt kann man nach diesem Modell annehmen, dass unabhängig von der Höhe des Faktors A der Effekt einer Veränderung von Faktor B der gleiche bleibt.

Wenn dieses Modell gilt, können die Effekte jedes Faktors separat mit einer Tabelle der Differenzen von einem „Basiswert“ beschrieben werden.

Es sollte festgehalten werden, dass dieses Modell nicht immer gilt. Zum Beispiel würde der Einfluss der Veränderung von Faktor A auf das Ergebnis wahrscheinlich geringer sein, wenn B auf einem niederen Niveau ist, als wenn B auf einem hohen Niveau ist. Dies wird Interaktion zwischen den Effekten von A und B genannt und daher muss ein anderes Modell angewendet werden. Interaktion wird später in diesem Kapitel behandelt.

Stärke von Asphaltbeton

Ein Ingenieur führte ein Experiment durch, um dein Einfluss der Verdichtungsmethode auf Asphaltbeton zu überprüfen. Dazu verwendete er zwei Gesteinsarten mit jeweils unterschiedlicher durchschnittlicher Fraktionsgröße und vier Verdichtungsarten in einem faktoriellen Versuch mit drei Wiederholungen.
