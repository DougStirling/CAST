Ordnen der Merkmalsträger

Viele wichitgen statistische Methoden nehmen an, dass die Merkmalsträger in einer Datenmatrix ungeordnet sind - eine Änderung in den Zeilen würde die selbe Information geben. Zum Beispiel: Die Gewichte von 20 Kühen einer Herde würden einen ungeordneten Datensatz bilden.

Aber manchmal sind die Zeilen der Datenmatrix geordnet, gewöhnlich durch Zeiten.Zum Beispiel: Temperaturwerte können in Zehn-Minuten-Intervallen zwischen 9:00 Uhr und 21:00 Uhr aufgenommen sein. Die resultierenden Temperaturen sind kontinuierliche, metrische (nummerische) Variablen, deren Messwerte zeitgeordnet sind. Das Ordnen der Messwerte gibt nützliche Hinweise, die einem helfen, die Daten zu verstehen. Diese Arten von Daten nennt man Zeitfolgen.

Für eine einleitende Untersuchung von geordneten Daten ist es oft nützlich, diese so zu begutachten, als wären sie ungeordnet. Allerdings sollte eine vollständige Analyse vor dem Ordnen der Daten gemacht werden.

Würde das Traubentest-Experiment in einer identischen Art für alle Trauben ausgeführt werden, wäre es möglich die Ordnung der Daten zu ignorieren und die Daten so zu Analysieren, als wären sie ungeordnet. 

Dies würde ebenso eine sinnvolle Anfangsanalyse der Daten des Wetter-Beispiels sein. Wenn die Daten aber über einen längeren Zeitraum gesammelt worden wären und diese einen Trend im Niederschlag zeigen würden (z.B. durch den Klimawandel), wäre es wichtiger das Ordnen der Werte zu beachten.
