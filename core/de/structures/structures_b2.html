Die meisten Variablen in einem Datensatz können durch einen von zwei Haupttypen klassifiziert werden.

Nummerische Variablen

Die Werte einer nummerischen Variablen sind Zahlen. Weiterführend können sie in diskrete und kontinuierliche Variablen klassifiziert werden.

Diskrete nummerische Variablen
Eine Variable, deren Werte ganze Zahlen sind, nennt man diskret. Zum Beispiel die Anzahl an Tomaten einer Tomatenpflanze ist diskret. 
Kontinuierliche nummerische Variablen
Eine Variable, die jeglichen Wert in einem bestimmten Bereich (auch Dezimalzahlen nach dem Komma) haben kann, nennt man kontinuierlich. Zum Beispiel das Gesamtgewicht von Tomaten, die an einer Tomatenpflanze wachsen, ist kontinuierlich.

Statistische Methoden, welche für kontinuierliche Variablen genutzt werden können, sind nicht immer auch für diskrete Variablen angebracht.

Die Unterscheidung zwischen kontinuierlichen und diskreten Variablen ist wichtig.

Eine diskrete Variable kann für gewöhnlich gezählt werden, wohingegen eine kontinuierliche Variable eine andere Art der Bemessung benötigt.

Kategorische Variablen

Die Werte einer kategorischen Variablen sind aus einer kleinen Gruppe von Kategorien ausgewählt. Beispiele sind Geschlecht (männlich oder weiblich) und Familienstand (ledig, verheiratet, geschieden oder verwitwet).

Kategorische Variable können außerdem in ordinale und nominale Variablen eingeteilt werden.

Ordinale kategorische Variablen
Eine kategorische Variable, wessen Kategorien sinnvoll geordnet werden können, nennt man ordinal. Zum Beispiel die Note eines Studenten in einer Prüfung (A, B, C oder Durchgefallen) ist ordinal.
Nominale kategorische Variablen
Es kommt nicht darauf an, auf welche Art die Kategorien in Tabellen und Grafiken geordnet sind. Alle Anordnungen sind gleichwertig. Zum Beispiel die Religion eines Studenten (Atheist, Christ, Muslim, Hindu, …) ist nominal.

Die meisten statistischen Methoden für kategorische Daten können für beide, ordinale und nominale Variablen, angewandt werden.

"Labels"

In manchen Datensätzen hat jeder Merkmalsträger einen eigenen "Namen", der zur Identifizierung genutzt werden kann. Diese Variablen nennen wir „Label-Variable“. Die Labels helfen uns, ungewöhnliche Beobachtungen in einem Datensatz zu identifizieren.

Warnung!

Manchmal sind kategorische Variablen, wenn die Daten erhoben werden, als Zahlen verschlüsselt (z.B. Geschlecht vielleicht verschlüsselt als 0 für9:00 männlich und 1 für weiblich). Die Variable ist immer noch kategorisch, trotz der Verwendung von Zahlen.

In gleicher Weise können die Merkmalsträger einer Befragung durch Zahlen verschlüsselt sein, welche sie einmalig identifiziert (eventuell aus Diskretion, um das Speichern von Namen in den Daten zu verhindern). Dies ist wirklich eine „Label-Variable“ und könnte einfach die Reihennummer in der Datenmatrix sein. 

Wenn du eine Spalte von Zahlen in deiner Datenmatrix siehst, gehe nicht unweigerlich davon aus, dass es sich um eine nummerische Variable handelt.

Charakteristika von Prostatakrebs-Patienten

Überprüfen sie den folgenden Datensatz. Er beschreibt die Charakteristika einer Gruppe von Patienten während der Aufnahme in ein Krankenhaus.

Name Alter Familienstand PSA im Blut Rauchen Tumorgröße
John Smith 54 verheiratet 45 nie Raucher gewesen klein
Mark Brown 65 verwitwet 37 derzeitiger Raucher mittel
Adam Jones 52 geschieden 66 ehemaliger Raucher mittel
Stuart Robertson 71 verheiratet 97 nie Raucher gewesen groß
... ... ... ... ... ...
  • Der Name ist eine "Label-Variable". 
  • Familienstand und Rauchen sind nominale, kategorische Variablen. (Wenn wir jedoch beachten, dass ehemaliger Raucher zwischen nie Raucher gewesen und derzeitiger Raucher liegt, dann kann es als ordinal behandelt werden.)
  • Tumorgröße ist eine ordinale kategorische Variable.
  • Alter und PSA im Blut sind kontinuierliche nummerische Variablen. (Obwohl die aufgezeichneten Alter zu ganzen Zahlen gestutzt wurden, ist die Benennung von Alter kontinuierlich.)
