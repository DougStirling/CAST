Ein lineares Modell für Mengenverhältnisse?

Würden wir versuchen, darzustellen, wie eine numerische unabhängige Variable eine numerische abhängige Variable beeinflusst, so würden wir eine lineare Gleichung verwenden, um die Beziehung zu veranschaulichen.

  =  b0 + b1 x

Ist die abhängige Variable kategorisch, so ist man versucht, eine ähnliche lineare Gleichung zu verwenden, um zu erklären, wie das Mengenverhältnis in einer abhängigen Kategorie von der unabhängigen Variable beeinflusst wird.

vorhergesagtes Mengenverhältnis,   

Unglücklicherwiese jedoch, ...

... ist eine lineare Gleichung nicht dafür geeignet, ein Mengenverhältnis darzustellen, sofern diese in vorhergesagten Mengenverhältnissen größer als 1,0 oder weniger als 0,0 ist.

Nichtlinear Modelle

Um zu veranschaulichen, wie ein Mengenverhältnis von einer numerischen unabhängigen Variablen, X, abhängig ist, sollte eine Gleichung Werte zwischen 0 und 1, für alle möglichen Werte von X, ergeben. Das heißt, die Gleichung muss nichtlinear in X sein.

Fruchtfliegen auf Mangos

Das Mengenverhältnis der überlebenden Eier wird im folgenden gestapelten Balkendiagramm gezeigt. Es wurde eine gerade Linie im Diagramm eingezeichnet, um zu verstehen, dass das Mengenverhältnis der sterbenden Eier von der Temperatur abhängig sein könnte.

Verschiebe die vertikale rote Linie auf der Achse, um das vorhergesagte Mengenverhältnis, der bei unterschiedlichen Temperaturen sterbenden Eier, zu erhalten.

Das lineare Modell ist eine einigermaßen genaue Angleichung an die Daten zwischen 39 und 45 Grad Celsius. Am Anstieg der Linie (ungefähr 0,056), können wir erkennen, das ungefähr 5 Prozent der Eier, bei einem jeweils weiteren Anstieg der Temperatur um 1 Grad Celsius, getötet werden.

Wie auch immer, sagt das lineare Modell vorher, dass mehr als 100 Prozent der Eier, bei Temperaturen über 46 Grad Celsius, getötet werden. Jedes lineare Modell wird Mengenverhältnisse, außerhalb des Bereichs von 0-1, für ausreichend extreme Werte von X, vorhersagen.

Wähle jetzt die Option nichtlineares Modell aus dem Pop-Up-Menü. Diese Funktion ist besser als die vorherige gerade Linie, da sie für alle Altersstufen, zwischen 0,0 und 1,0 liegt.

Verschiebe die vertikale rote Linie nochmals auf der Achse, um das vorhergesagte Mengenverhältnis, der, jeweils bei unterschiedlichen Temperaturen, sterbenden Eier, zu erhalten. Ein nichtlineares Modell kann bei allen Temperaturen gute Vorhersagen liefern.
