Computer können gute Werte für die Konstanten der Modelle finden

Lineare Modelle werden dadurch an Daten angeglichen, dass die Werte der zwei Parameter b0 und b1 ausgewählt werden, um die Summe der Quadrierten Residuen zu vermindern.

Leider können die Parameter b0 und b1 eines logistischen Modells, mit einem so einfachen Kriterium, nicht erhalten werden. Die Angleichung der Modelle von Mengenverhältnissen, basiert auf einer Methode namens Maximum-Likelihood, welche über den Bereich von CAST hinausgeht..

Wie auch immer gibt es viele statistische Programme, die entsprechende Berechnungen für dich durchführen. Daher verwenden wir eine Black-Box-Methode und zeigen, was die Parameter-Einschätzung ohne eine weitere Justierung ergibt.

Das unten angeführte Diagramm zeigt die Daten der überlebenden Fruchtfliegeneier.

Verschiebe die zwei roten Pfeile auf der logistischen Funktion, um die Parameter der Funktion zu verändern. Versuche die Funktion so gut wie möglich an die Daten anzupassen.

Klicke schließlich auf die Taste Beste Angleichung, um die „besten“ Werte für die Parameter zu sehen.
