Kausale Zusammenhänge

In vielen bivariaten Datensätzen ist die Beziehung zwischen zwei Variablen nicht symmetrisch. Von der Art der Variablen und der Weise wie die Daten gesammelt wurden, mag es klar sein, dass eine Variable X potenziell Einfluss auf eine andere Variable Y nehmen kann, aber, dass der Umkehrschluss unmöglich ist.

In solchen Daten wird die Variable X unabhängige Variable und Y abhängige Variable (Antwort) genannt.

Experimente

Ein Experiment ist eine Art von Datenerhebung, in der die Person, die das Experiment durchführt, die Werte der unabhängigen Variablen kontrolliert- z.B. die Anpassung der Temperatur für jede Durchführung des Experiments, wie im folgenden, eingerahmten Beispiel. Ein gut konstruiertes Experiment stellt immer sicher, dass die Beziehung zwischen der unabhängigen und abhängigen Variablen ursächlich (kausal) ist.

CO-Produktion und Temperatur

Wenn wir den Betrag an Kohlenmonoxid (Y) messen, der bei einer Reaktion, die bei unterschiedlichen Temperaturen (X) durchgeführt wird, produziert wird, dann kann die CO-Produktion von der Temperatur abhängig sein. Aber die Temperatur ist nur vom Experimentierenden festgelegt.

Beobachtungsstudien 

Viele Datensätze werden nicht durch Experimente gewonnen. Wenn die Person, die die Daten sammelt, keine Kontrolle über die Variablen hat und einfach ein Paar Werte von jedem Merkmalsträger aufzeichnet, dann werden die Daten Beobachtungen genannt. Wenn es eine zeitliche Reihenfolge von zwei Variablen in einer Beobachtungsstudie gibt- eine Variable ist eine frühere Beobachtung als die andere– dann kann man auch in der Lage sein die Beziehung kausal zu behandeln, mit der späteren Variablen als Antwort.

Auch wenn eine Beziehung nicht kausal ist, ist man manchmal an der Vorhersage des Wertes einer Variablen, die durch eine andere Variable beeinflusst wird, interessiert. In dieser Situation würde man die Daten der vorhergesagten Variablen, die man als die abhängige Variable ansieht, analysieren. 

Körpergrößen von Vätern und Söhnen

In einem Datensatz, der die Größen von Vätern und deren 18jährigen Söhnen beinhaltet, kann man behaupten, dass die Größen der Väter X, die Größen der Söhne Y durch genetische Vererbung beeinflussen kann aber, dass der Einfluss nicht in der umgekehrten Richtung stattfinden kann. 

Körperfett und Hautfaltendicke 

Präzise Messungen des Körperfetts Y sind schwierig machbar, wohingegen Messungen der Hautfaltendicke X relativ einfach gemacht werden können. Es wäre nützlich, wenn man in der Lage wäre das Körperfett durch die Hautfaltendicke, basierend auf Datenmaterial aus beiden Messungen von einer Gruppe von Leuten, vorherzusagen. Man könnte in einer Analyse der Daten dann Körperfett als abhängige und Hautfaltendicke als unabhängige Variable ansehen. 

Bevor man bivariate Daten analysiert,

sollte man immer berücksichtigen, ob eine Variable als Antwort angesehen werden sollte. 

Einige statistische Methoden kann man nur nutzen, wenn solch eine Klassifizierung der Variablen vorgenommen wurde.
