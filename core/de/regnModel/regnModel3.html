Normales lineares Model für die Antwort

Ein Regressionsmodell für die Antwort in einem bivariaten Datensatz beschreibt, wie die unabhängige Verteilung von X abhängt. Das am häufigsten verwendete Regressionsmodell ist ein normales lineares Modell. Dieses Model beinhaltet:

Normalität
Bei jedem Wert von X hat Y eine normale Verteilung.
Konstante Varianz
Die Standardabweichung von Y ist die gleiche für alle Werte von X.
Linearität
Der Mittelwert von Y steht in einem linearen Verhältnis zu X.

Die letzten beiden Anteile des normalen linearen Modells können wie folgt ausgedrückt werden

σy  =  σ

μy  =  β0  +  β1x

Beachte: nur die Antwort ist modelliert

Ein normales lineares Model versucht nicht die Verteilung von x-Werten zu erklären.

Experimentelle Daten
Bei experimentellen Daten sind die Werte für X durch den Experimentator festgelegt, ihre Verteilung ist also uninteressant.
Beobachtete Daten
Bei beobachteten Daten sind die Werte für X meist zufällig. Obwohl es auch möglich ist diese auch mit einer univaraiten Verteilung zu modellieren wird dies nur selten getan. Das Verhältnis zwischen X und Y wird mit einem Regressionsmodell analysiert, welches die einzelnen Werte von X als Konstanten behandelt. Es ist wichtig zu verstehen, dass das Regressionsmodell nur die abhängige Verteilung von Y bei jedem X beschreibt.

Beispiel für ein normales lineares Modell

Unterhalb wird ein normales lineares Modell gezeigt.

Man kann den Schieberegler bewegen, um zu sehen, wie die Verteilung von Y von den Werten von X abhängt.

  • Die grüne Linie in dem Diagramm ist das Zentrum der unabhängigen Verteilung, Regressionsgerade genannt. In diesem Beispiel wird sie durch die Gleichung beschrieben

    μy  =  2.5  +  1.5x

  • Die Streuung der unabhängigen Verteilung ist für alle X gleich,

    σy  =  0.8

Wenn man ein paar mal auf Stichprobe ziehen klickt, können typische Daten für dieses Model  beobachtet werden, wenn 5 unabhängige Messungen für jedes X = 1,2,3 und 4 gemacht worden sind.

Dieses Model kann auch in Situationen genutzt werden, bei denen die Werte für X nicht wiederholt werden. Wählt man die Option reguläres X und nimmt dann einige Proben mehr, kann man typische Daten sehen, wenn die Werte für X 0,6; 0,8; 1,0;...;4,4 sein sollen.

Wähle die Option zufälliges X und nehme einige Proben mehr, um typische Daten zu sehen, wenn die Werte für X ungleichmäßige Abstände haben.
