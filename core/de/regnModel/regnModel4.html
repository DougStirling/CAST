Beschreibung des Models im Fall einer unabhängigen Verteilung

Das normale lineare Model beschreibt die Verteilung von Y für jeden Wert von X. Ausgedrückt werden kann dies in der Form...

Y   ~   normal (µy, σy)

wobei

µy   =   β0   +   β1 x

σy   =   σ

Beschreibung des Models unter Bedingungen von „Fehlern“

Eine gleichwertige Art dieses Model zu beschreiben ist...

y   =   β0  +   β1 x  +  ε

wobei ε der Modellfehler genannt wird und folgende Verteilung hat

ε  ~   normal (0, σ)

Es ist hilfreich zu beobachten, dass der Fehler ε, für einen Datenpunkt...ist.

ε   =  y  −  (β0  +   β1 x)

welcher den vertikalen Abstand zwischen einem Kreuz in einem Streudiagramm und der Regressionsgerade beschreibt.

Hier lohnt es sich zu betonen, dass in praktischen Situationen,

Der Anstieg und der Achsenabschnitt der Regressionsgerade, ß0 und ß1, unbekannte Parameter sind. Die Fehler, ε, können daher nicht genau bestimmt werden.

Im nächsten Abschnitt werden wir sehen, wie diese Mengen abgeschätzt werden können.

Bandbreite mit über 95% der Werte

Die 70-95-100-Regel gibt an, dass etwa 95% der Werte in einer beliebigen Probe innerhalb von 2 Standardabweichungen des Mittelwertes liegen. Im Zusammenhang mit einem normalen linearen Model werden daher etwa 95% der Fehler innerhalb von 2 Standardabweichungen von Null liegen – z.B. zwischen ±2σ.

Da die Fehler die senkrechten Abstände von Punkten von der Regressionsgerade sind, bedeutet dass...

Etwa 95% der Kreuze werden innerhalb ±2σ der Regressionsgeraden (senkrecht) liegen.

Daher gibt es ein Band 2σ auf jeder Seite der Regressionsgeraden, das etwa 95% der Kreuze in einem Streudiagramm von Daten enthält.

Beispiel

Das Diagramm unten zeigt ein normales lineares Model mit den Parametern

μy  =  2.5  +  1.5x

σy  =  0.8

Die blauen Bereiche in den Ausläufern der normalen Wahrscheinlichkeitsdichten-Funktion sind größer als 2σ (z.B. 1,6 für dieses Model) auf jeder Seite von µy. Sie sind etwa 5% des Normalverteilungsbereiches, so das rund 95% der y-Werte gesammelt von dieser Verteilung innerhalb dieser Grenzen liegen. Dies trifft für jedes X zu – um dies nachzuprüfen kann man den Schieberegler bewegen – also werden etwa 95% der gesammelten Werte in dem grauen Band der x-y Ebene liegen.

Um nachzuprüfen, dass etwa 95% der Werte innerhalb des grauen Bandes liegen, klickt man einige Male auf Stichprobe ziehen.

Abschließend klickt man auf den Schalter oben rechts in dem Diagramm, um auf die x-y Ebene zu blicken. Auf späteren Seiten werden wir ein normales lineares Model mit einem 2-dimensionalen Diagramm dieser Form vorstellen.
