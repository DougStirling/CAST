Linear transformations

Many types of measurement (e.g. temperature, weight and distance) can be expressed in different units that are linearly related.

The exercise below gives a data set whose values are recorded and displayed in some units (e.g. kilometers). The question ask for the proportion of values that are less than some cutoff value in different units (e.g. miles).

  • Use one of the two templates to change the units of the cutoff value.
  • Then drag the vertical red line on the stacked dot plot to count the number of values below the cutoff.
  • Finally evaluate the proportion of values and type it into the answer box.

Repeat with a few different questions.
