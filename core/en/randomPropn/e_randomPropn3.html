In the exercise on this page, you will use the computer to find probabilities relating to variables with binomial distributions.

Use the sliders to set the parameters of the distribution, then drag the vertical red lines to highlight the bars for the binomial counts to which the question refers. Finally type the displayed probability into the answer box.

(Read the wording of the question carefully! It is easy to incorrectly include or exclude the value at one end of the interval.)

Repeat with a few questions until you are confident with how to obtain binomial probabilities.
