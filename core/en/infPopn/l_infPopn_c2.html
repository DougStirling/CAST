Pages printed per ink jet cartridge

Use this simulation to demonstrate that repetition of the data collection would usually result in different values. The point again is that a single sample is not as important as the 'process' behind the data — i.e. we want to generalise from the sample to something more fundamental about the number of pages printed by a print cartrige in general.

This is a simulation but the context is an experiment to assess the number of pages that can be printed by a new range of cartridges for an ink jet printer.
