Cluster sampling

Use this diagram to show that cluster sampling results in less accuracy (when estimating a mean) if the values are similar within clusters. Each column of 4 values in the diagram corresponds to a cluster.

With the slider to the left (high variability within clusters), simple random and cluster sampling have similar accuracy — the sample means have similar spread.

With the slider to the right (all values same within clusters),

  • Cluster sampling is much less accurate than simple random sampling.
  • The sample size is effectively 5 values (clusters) from 24, not 20 values from 96, so the usual formula for the standard deviation overestimates accuracy.
