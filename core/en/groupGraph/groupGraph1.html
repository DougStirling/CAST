Comparisons

We often want to make comparisions. Examples of things that we might want to compare are:

  • Different countries or regions within one country
  • Different groups of individuals or households (e.g. urban and rural)
  • Different years

These comparisons are often based on a collection of values from each 'thing', arranged in a two-way table in which one dimension (often rows) corresponds to the different groups. We will restrict attention to two-way tables in which the values all describe related values. Examples are:

  • Expenditures in different categories
  • Counts of individuals of different types (forming a frequency table)
  • Average lengths of prison sentences

Terminology

Although the terminology does not perfectly fit all examples, we will call the main things being compared groups. The related values from each group will be called measurements in different categories.

   Group    Measurement    Categories   
Region Number of households Household size
Age group Number of prisoners Average length of sentence
Country Life expectancy Gender
Year Expenditure Expenditure category

Note that the values in the first two examples for a frequency table for each group.

Graphical display

A simple bar chart can be drawn to display the values in each group. However if there are many groups, it is difficult to draw them close enough to each other to effectively compare them. We will describe some other graphical displays later in this section.
