Clicking an icon on the right of the banner makes it easier to print information about the current e-book.

List of pages

For any chapter, you can display a heirarchical list of its sections and pages in a separate browser window. By default, a line or two of summary about each page is displayed, but a check-box allows these summaries to be hidden.

This list can either be printed directly from the browser or copied and pasted into a word processor such as Microsoft Word for further formatting.

Summary pages

If the current e-book contains summary versions of its pages, it is possible to display the complete summary version of any chapter in a single page that is formatted for printing.

You can print directly from the browser, but for a few e-books, we have prepared PDF versions of these summary chapters on our server that are laid out for printing 2-pages per sheet of paper.

Printing the summary pages

Many of the CAST e-books contain summary versions of their pages. For the public introductory e-books, these summary pages have been compiled into PDF files in a 4-per-page format for printing. This has also been done for some customised e-books.

If PDF versions of the book's summary chapters have been created, a pop-up menu on the right of the banner will give you access to them.

If you select one of these chapters from the menu, its summary PDF will be displayed in your default web browser and can be downloaded or printed from there.
