Applying an incomplete design

There are two steps to the application of an incomplete design in an actual experiment.

Treatments to use
The first decision relates to the choice of treatments to use (and not use), either overall in a completely randomised experiment or within each block of a randomised block experiment.
Randomisation
After the treatments have been picked, they must be randomly allocated to physical experimental units, either overall or within each block.

Picking treatments

In a Latin square design, there are many different ways to write letters in a square grid so that each appears in exactly once in every row and column. Randomisation

Random allocation of treatments to physical experimental units depends on whether there are blocks in the experiment.

Rows and columns are both blocking variables
No further randomisation is needed — each cell of the table is effectively a separate block so it only has a single treatment.
Rows are blocks
The treatments (column-letter combinations) should be randomly allocated within each block.
No blocks
In a completely randomised experiment, all treatments (row-column-letter combinations) should be randomly allocated to the experimental units.
