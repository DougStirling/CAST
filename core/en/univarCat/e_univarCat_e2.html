A pie chart displays the same information as a bar chart but in a different format.

This exercise is similar to the one on the previous page. However in this case, the individuals belong to four different groups (with a single categorical measurement from each individual) so the data are exported in 'stacked' format -- one column contains the categorical values and the second column contains the names of the groups.

You should again export the data and draw pie charts for the four groups of individuals; this will allow you to match each pie chart with the name of its data by dragging the pie charts up or down.
