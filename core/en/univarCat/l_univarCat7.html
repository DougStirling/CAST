New Zealand wine production

The three variables are all quantities, so they can be displayed in bar charts.

However the 'category' axis for this data set is 'year' so an alternative display is a standard time series plot.

You could mention that production per hectare (production divided by area) highlights a different aspect to the data from the separate plots of production and area — sometimes a simple ratio or difference between variables is more meaningful.

The data set contains information about wine production in New Zealand from 1961.

Both wine production and area increased over this period. However production per hectare has decreased. This could have been caused by starting to grow wine on poorer soils or by focusing on higher quality (but less productive) varieties.
