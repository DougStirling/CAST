Warning about over-interpreting

Adjusting the class width and the starting position for the first class causes a surprising amount of variability in histogram shape for small data sets, so be very wary of over-interpreting features such as clusters or skewness.

Yam growth data

Dot plots should be used in preference to histograms for small data sets.

Because a stacked dot plot shows the size of a data set, it gives some warning about the risk of over-interpreting the distribution of a small data set.
