Use the Wider and Narrower buttons to show that:

  • When class width is less than 4.0, the histogram starts to look jagged
  • When class width is greater than 8.0, the histogram becomes blocky and shape information between 0 and 10 is lost by the grouping.

Any class width between 4.0 and 8.0 would be acceptable.

Use the Left and Right buttons to show that the starting-point for the first class also affects histogram shape but is less important than the class width.

Mention that if your conclusion from a histogram depends on the choice of classes, you are over-interpreting it.

The data set shows 200 values which are concentrations of a pesticide in Golden Delicious apples (parts per million).
