Any type of hypothesis test

Use this diagram to explain that for any type of hypthesis test,

  • The cut-off for accepting/rejecting H0 can be based on the p-value.
  • The significance level of the decision rule (prob of Type I error) equals the cut-off for the p-value.

Use the top slider to adjust the cut-off. Mention that:

If computer software provides the p-value for a hypothesis test, it is easy to translate it into accept (or reject) the null hypothesis at the 5% or 1% significance level.
