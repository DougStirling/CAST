Normal linear model

μy  =  2.5  +  1.5x         σy  =  0.8

The normal linear model on the previous page was for experimental data in which there were only a few x-values used. This diagram shows that a normal linear model specifies a normal distribution for the response at all x.

Drag the slider to show how the response distribution depends on x.

Click the y-x rotation button then take samples from the model. Depending on the pop-up menu, there can be repeated response values at four x-values, regularly spaced x-values, or random x-values for the data. Explain that the normal model covers all of these.
