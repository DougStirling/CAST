Information can be obtained from multi-level data by examining both the group-level and individual-level data matrices.

Household survey in Malawi

This e-book only deals with analysis of a single data matrix. State that information in multi-level data can be transferred from one level to the other:

  • Top-level data can be copied down to the sub-level data matrix
  • Information from the sub-level data matrix can be summarised in the top-level data matrix.

Select variables from the pop-up menu to see whether they are copied from the top-level data matrix to the sub-level matrix or summarised from the sub-level data matrix to the top-level one.

Explain that properly analysing multi-level data can be tricky:

These data come from a survey of households in Malawi. For illustrative purposes, we have only shown data from 18 households and only a few variables.

  • Household size, minimum age and average age summarise information from the sub-data-matrix of individuals to household level.
  • Distance to water is copied from household level to individual level.
