Stacking the crosses shows density better

The randomness of the jittering in a jittered dot plot can be disconcerting. Another way to separate the crosses that shows high and low density of values better is with a stacked dot plot.

Jittered vs stacked dot plots

When analysing real data sets,

Stacked dot plots show the distribution of values in a data set better than jittered dot plots.

However jittered dot plots are better when explaining some statistical concepts, so it is important that you understand them.

Jittered dot plots are extensively used in CAST.
