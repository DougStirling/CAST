Drawing a stacked dot plot

In practice, you should always rely on a computer to draw dot plots.

However the following exercise asks you to complete stacked dot plots 'by hand' to check that you understand how they are produced (and therefore interpreted).

A list of values is shown on the left. The stacked dot plot on the right displays the data except for the final two values on the list, represented by crosses at the top of the diagram. Drag these to the correct stacks of the dot plot. (When you select any cross, its value is highlighted in the list.)

If you have any difficulty, repeat with a few different data sets.
