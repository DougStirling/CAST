Splitting values into stems and leaves

Dragging the slider splits each value in the list into a stem and leaf (and possibly digits that are discarded).

The three examples are of increasing difficulty:

  • In the data set Multi-digit stems, the decimal point is not included in the stems.
  • In the data set Values that must be truncated, some digits from each value are ignored.

Mention that between 10 and 20 stems usually gives a good stem and leaf plot.
