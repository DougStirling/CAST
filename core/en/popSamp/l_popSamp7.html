Random sample without replacement

Each population member must be given a number — 0 to 55 here.

Click Random index several times to generate random numbers between 0 and 99.

  • Random numbers outside the range 0 to 55 must be ignored.
  • If the individual is already in the sample, another random index must be generated — we don't want the same individual twice. This happens more often as the sample size increases.

Click Faster to generate indices faster.
