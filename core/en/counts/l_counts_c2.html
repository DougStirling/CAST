Computers purchased by university departments

Histograms are good displays of data sets containing large or moderate counts, such as attendance figures at football matches. (If all counts are small, bar charts are better.)

In this data set, it could be argued that a bar chart would be better since there are only 38 possible values (0 to 37). However by reducing class width to 1, the diagram does allow you to explain that:

  • class widths should be whole numbers
  • class boundaries should end with '.5' — when class width is 1, classes are centred on the counts

A class width of 3 gives a reasonably smooth picture of the distribution here.

The data set contains the number of recently purchased computers (under 2 years old) in each of the 106 departments of a large university.
