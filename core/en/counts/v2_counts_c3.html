Displaying small counts

When the counts in a data set are small, a histogram would be drawn with class width 1 and the classes centred on 1, 2, etc. If the rectangles are narrowed so that they do not touch, the histogram is called a bar chart of the data. A bar chart describes the distribution better since it emphasises the discrete nature of the data.
