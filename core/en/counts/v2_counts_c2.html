Displaying moderate or large counts

Histograms can be used instead of dot plots for discrete variables. Class boundaries should end in '.5' to ensure that data values do not occur on the boundary of two classes.

In some data sets, all values are large counts. For example, attendance records at professional football matches would all be greater than 1,000. If all counts are large, the precise location of class boundaries is less important.
