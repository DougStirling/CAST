La mayoría de las variables en un conjunto de datos se pueden clasificar en uno de dos tipos importantes.

Variables numéricas

Los valores de una variable numérica son números. Pueden ser clasificados más a fondo en variables discretas y continuas.

Variable numérica discreta
Una variable cuyos valores son números enteros (cuentas) se llama discreta. Por ejemplo, el número de días con lluvia en un año es discreto.
Variable numérica continua
Una variable que puede contener cualquier valor dentro de una cierta gama se llama continua. Por ejemplo la precipitación anual total es continua.

Los métodos estadísticos que se pueden utilizar para las variables continuas no son siempre apropiados para las variables discretas.

La distinción entre las variables discretas y continuas es importante.

Una variable discreta se puede identificar generalmente como siendo una cuenta, mientras que una variable continua es una cierta otra clase de medida.

Variables categóricas

Los valores de una variable categórica se seleccionan de un pequeño grupo de categorías. Los ejemplos son género (varón o hembra) y estado civil (nunca casado, casado, divorciado o viudo).

Las variables categóricas se pueden categorizar más a fondo en variables ordinales y nominales.

Variable categórica ordinal
Una variable categórica cuyas categorías pueden ser pedidas significativo se llama ordinal. Por ejemplo, el grado de un estudiante en un examen (A, B, C o fall) es ordinal.
Variable categórica nominal
No importa que la manera las categorías se pide en las exhibiciones tabulares o gráficas de los datos — todos los orderings son igualmente significativos. Por ejemplo, religión de un estudiante (ateo, cristiano, musulmán, hindúes,…) es nominal.

La mayoría de los métodos estadísticos para los datos categóricos se pueden aplicar a las variables ordinales y nominales.

Distinguimos raramente entre las variables ordinales y nominales en MOLDE.

Etiquetas

En conjuntos de datos ciertos, cada uno individual tiene un “nombre único” que se pueda utilizar para identificarlo. Llamamos tal variable una variable de etiqueta. Las etiquetas pueden ayudarnos a identificar observaciones inusuales en el conjunto de datos.

¡Advertencia!

Las variables a veces categóricas se cifran como números cuando se registran los datos (e.g. el género se puede cifrar como 0 para los varones y 1 para las hembras). La variable es todavía categórica, a pesar de el uso de números.

De una manera similar, los individuos en un examen pueden ser cifrados con un número que los identifique únicamente (quizás evitar almacenar nombres en los datos para el secreto). Esto es realmente una variable de etiqueta y puede ser simplemente el número de la fila en la matriz de los datos.

Cuando usted ve una columna de números en su matriz de los datos, no asuma que es una variable numérica.

Características de empleados

Considere el conjunto de datos siguiente que describe las características de los empleados de una compañía.

Nombre Sexo Edad Estado civil No de niños Renta El fumar
John Smith varón 24 solo 0 $25.000 nunca ahumado
Maria Brown hembra 35 casado 3 $45.000 fumador actual
Adán Jones varón 42 divorciado 1 $40.000 fumador anterior
Jane Robertson hembra 29 divorciado 0 $42.000 nunca ahumado
… … … … … …
  • El nombre es una variable de etiqueta.
  • El sexo, el estado civil y el fumar son variables categóricas nominales. (Sin embargo si miramos al “fumador anterior” como estando entre “fumador nunca ahumado” y el “actual” entonces podría ser tratado como ordinal.)
  • La edad y la renta son variables numéricas continuas. (Aunque las edades registradas se hayan truncado a los números enteros, el concepto de edad es continuo.)
  • El número de niños es una variable numérica discreta (una cuenta).
