Valores cabidos

Un modelo linear,

y  =  b0 + b1 x

implica dos constantes (la cuesta y la intercepción) cuyos valores sean desconocidos; se llaman los parámetros desconocidos del modelo. ¿Cómo debemos fijar sus valores para emparejar un conjunto de datos particular?

Para determinar como de bien los ajustes particulares de un modelo linear de nuestros puntos de referencias, (xi, yi), nosotros pudieron considerar como de bien el modelo prediría el y-valor del punto,

 =  b0 + b1 xi

Estas predicciones para los x-valores en el conjunto de datos se llaman los valores cabidos.

Residuales

La diferencia entre los valores cabidos para un punto de referencias y su y-valor real se llama la residual del punto.

Las residuales describen los “errores” que habrían resultado de usar el modelo para predecir la respuesta para los x-valores en los datos.

Temperatura y latitud

El scatterplot abajo demuestra la temperatura máxima de enero (y, los grados Fahrenheit) y la latitud (x, grados) de varias ciudades en los E.E.U.U.

Una línea que podemos considerar como modelo linear para estos datos se dibuja en gris en el diagrama.

Chasque encendido las cruces individuales para determinar las ciudades de las cuales las medidas fueron hechas. Los valores y las residuales cabidos también se exhiben sobre el scatterplot. Por ejemplo, Seattle Tacoma, una de las cruces sobre la línea en la derecha inferior, tiene una temperatura real de 44 grados (la posición vertical de la cruz), una temperatura cabida de 33.18 grados, y una residual de 10.82.

Observe que las residuales son las distancias verticales de las cruces a la línea.
