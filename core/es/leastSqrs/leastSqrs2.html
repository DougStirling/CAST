Formas matemáticas simples para la línea de regresión

Aunque cualquier forma de la línea de regresión se pueda dibujar “por el ojo” a través de un scatterplot, es conveniente restringir la atención a las funciones matemáticas simples

y  =  ƒ ( x )

Usando un modelo de este tipo, podemos predecir la respuesta para cualquier valor, x, de la variable explicativa insertando ese valor en la función — mucho más fácil que la lectura de una predicción de una curva que es dibujada “por el ojo”.

Ecuaciones lineares

Aunque algunas relaciones se deban describir por las curvas, una línea recta es una descripción adecuada de muchos conjuntos de datos de dos variables. Una línea recta corresponde a una ecuación de la forma

y  =  b0 + b1 x

El b0 constante se llama la intercepción de la línea y describe el y-valor cuando el x-valor es cero. El b1constante se llama la cuesta de la línea; describe el cambio en y cuando x aumenta en uno.

Cuando está utilizada para describir una relación entre dos variables, una línea tan recta se llama un modelo linear o un modelo de regresión linear.

Las predicciones se hacen fácilmente de un modelo linear

Un modelo linear se puede utilizar para predecir el valor de y que sería obtenido en cualquier valor de x, usando la ecuación

  =  b0 + b1 x
