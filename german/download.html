Vorteile herunterladen

Obwohl CAST können direkt von der genutzt werden ?????? Website, ermutigen wir Anwender CAST herunterladen und starten Sie es von einer lokalen Festplatte.

  • CAST kann langsam sein, wenn aus schlechte Internet-Anschlüsse laufen.
  • Laufen CAST von einer lokalen Kopie reduziert die Belastung auf die ?????? Web-Server.
  • Eine heruntergeladene Kopie kann auf einem Computer ohne Internet-Verbindung verwendet werden.

Noch wichtiger ist, dass die interaktive und dynamische Diagramme, die ein Kernbestandteil von CAST sind noch nicht von einigen Web-Browsern unterstützt und wird schließlich aufhören, in anderen arbeiten.

Herunterladen und Installieren

Der Deutschen Sammlung von CAST e-Bücher wird als komprimiert "zip"-Datei zur Verfügung gestellt.Klick auf diesen Link eine Datei namens "cast_for_germany_installer.zip" herunter. Sie sollten jetzt diese Datei aus Ihrem Standard-Download-Ordner, wo Sie wollen CAST installiert werden kopiert — Desktop bietet wohl die bequemste Ort sein.

Wie CAST extrahieren aus dieser Datei ist abhängig von Ihrem Betriebssystem.

Windows
Rechtsklick auf die ZIP-Datei und wählen Sie dann den Befehl Extract alll im Popup-Menü, das erscheint. Sie werden gefragt, wo Sie wollen, installiert CAST. Dadurch entsteht ein Ordner mit dem Namen CAST auf der Festplatte, die alle CAST. (Beachten Sie, dass Sie einfach einen Doppelklick auf die ZIP-Datei werden die Dateien, die in ihm sind Ihnen zeigen, wird aber nicht in einer Form, die CAST verwendet werden können extrahieren Sie sie — müssen Sie den Befehl verwenden Extract all.)
Macintosh
Auf einem Mac, doppelklicken Sie einfach auf die ZIP-Datei; CAST wird in einem Ordner namens CAST in der gleichen Position wie der Zip-Datei erweitert werden.

In beiden Fällen werden Sie mit einem Ordner namens CAST auf der Festplatte verbleiben. Die ZIP-Datei selbst kann nun gelöscht werden.

Java

Sie benötigen Java-Unterstützung auf Ihrem Computer zu installieren, sowohl die Standalone -Version von CAST zu laufen und auch den Zugriff auf die interaktive Diagramme im Browser-basierte version.You sollte Oracle die wichtigsten Java-Website gehen zu bekommen und stellen Sie sicher, dass Sie haben die neueste Version von Java installiert ist.

Laufende CAST von einem heruntergeladenen Kopie

Auf der obersten Ebene Ihrer installierten CAST-Ordner, sollten Sie eine Datei namens "Start_CAST.jar" zu sehen. Wenn die Java-Unterstützung installiert wurde, einen Doppelklick auf diese Datei sollte die Standalone -Version CAST starten.

Wenn Sie den Browser-basierte Version von CAST behalten möchten (nicht empfohlen), können Sie einen Doppelklick auf die Datei "collection_german.html"; dies sollte diese Version von CAST in Ihrem Standard-Browser anzeigen. (Es wird dem Fenster identisch sein, die Sie sich gerade befinden, es sei denn es eine neuere Version von CAST ist).
